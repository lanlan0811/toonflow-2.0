# 《紫霄浮岛·七日等候》导演手册

本导演手册专用于 `data/shot-list/01_storyboard.md` 的 82.8 秒片段，属于“东方仙侠 / 宗门群像 / 轻冲突 / 悬念式结尾”。它与 `art_skills/3D_xianxia_zixiao_waiting` 视觉手册配套使用：本手册决定叙事、调度、节奏、声音和信息揭示；视觉手册决定角色、场景、道具和渲染的一致性。

## 一句话导演命题

把“等不到一个人”拍成一场有层次的群像戏：先让观众期待回应，再用空岛证据制造悬念，用三种等待态度制造轻冲突，最后用七天的静止把虾仁的缺席变成下一段剧情的钩子。

## 片段结构

| 段落 | 镜号 | 时长功能 | 观众获得的信息 | 导演重点 |
|---|---|---|---|---|
| 建立地点 | 01—04 | 约 15.5 秒 | 四人来到紫霄剑宗浮岛；岛内没有回应 | 空间轴线先清楚，静默逐步放大 |
| 旧怨与判断 | 05—10 | 约 26.6 秒 | 红衣师妹积怨，夏浅浅发现岛上像无人 | 用反应和环境证据替代解释堆砌 |
| 决策与分头 | 11—16 | 约 32.7 秒 | 墨秋霜决定等，南宫离外出寻找，三人留守 | 南宫离的离场是唯一短促动作峰值 |
| 七日悬念 | 17—18 | 约 8.0 秒 | 三人真的等了七天，虾仁仍未现身 | 以时间流逝收尾，不揭示答案 |

## 绝对事实

- 虾仁不出镜；不得用任何人物形态补足他的缺席。
- 岛上不出现杂役、弟子、群众或无关生物群；“无人”是悬念的证据。
- 南宫离在第 14 镜离开，之后不再回到画面；第 17—18 镜只剩墨秋霜、夏浅浅、红衣三师妹。
- 结尾是七天等待，不是虾仁归来或冲突解决。
