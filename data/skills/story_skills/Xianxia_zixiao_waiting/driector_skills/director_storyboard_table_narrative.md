# 分镜表叙事手法 · 《紫霄浮岛·七日等候》

## 一、镜头编排原则

分镜表必须服务“期待回应 → 发现空岛 → 争执与折中 → 七日等待”的信息顺序。每个镜头至少完成一项任务：交代空间、给出人物反应、补充无人证据、推动决策或延长悬念。相邻镜头若承担同一任务，应优先合并，不用碎切制造假节奏。

## 三、景别递进与时长

- 01—04 以全景和中远景为主，允许意境镜头接近 6—8 秒，但画面必须有云、雾、回声或衣摆的自然动态。
- 05—10 将景别收至中景和中近景，给表情留呼吸，不连续切换同一轴线的正反打。
- 11—16 以中近景和群像为主，14 镜短促、其他镜头保持沉稳，形成一次而非多次动作峰值。
- 17—18 回到全景，时长服务“等待很久”的体感；最后一镜保留空旷负空间。

## 四、转场与声音

- 01→02：同一空间轴线的机位推进；不要突然换建筑。
- 03→04：回声衰减后切到空岛，静音是转场点。
- 06→07：从红衣情绪切到夏浅浅的观察，利用视线方向匹配剪辑。
- 13→14：夏浅浅提出方案后，以南宫离抬手/玉牌声作动作接点。
- 16→17：墨秋霜的收束台词落下后，用姿态相同的叠化进入等待。
- 18：低沉钟声或箫尾音收束，淡出黑场；不用花式转场、闪白或字幕卡。

## 五、分镜表终检

1. 每一镜是否写明谁在画面中、谁已离场？
2. 是否保留“空岛无人”的空间证据？
3. 是否有任何镜头暗示虾仁出现？
4. 南宫离离场后是否只剩三人，并且没有错误回到画面？
5. 角色动作是否符合古风仪态且可由 3D 模型稳定执行？
6. 场景五视图所锁定的建筑轴线是否贯穿全片？
7. 结尾是否停在“七天等待”的悬念，而非提前给出答案？
