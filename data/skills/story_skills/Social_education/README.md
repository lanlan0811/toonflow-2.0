# 社会教育 · 导演叙事手法技能包

## 简介

本技能包为 **社会教育** 类型故事提供一套完整的导演叙事手法参考，涵盖从宏观叙事规划到微观分镜执行的全流程指导。适用于任何视觉风格。

## 核心理念

- **真实即力量** — 社会教育短片的力量不在于口号喊得多响，而在于让观众相信"这是可能发生在自己身上的事"。真实感是教育说服力的基础
- **让观众自己"发现"道理** — 最有效的教育不是告诉观众"什么是对错"，而是让观众跟随角色的经历，自己得出"原来这是错的"的结论。体验式叙事优于说教式灌输
- **60秒情感过山车** — 社会教育短片的黄金时长。需要在60秒内完成"好奇→担忧→紧张→醒悟→坚定"的完整情感闭环
- **关键信息视觉化** — 核心教育信息（法律条文/危害后果/求助方式）不应靠台词念出，而应通过屏幕文字/文件/横幅/图标等画内视觉元素呈现，让信息自己"说话"
- **家属/亲友视角的叙事力量** — 社会教育中，受害者的家人/朋友作为"干预者"登场，比直接说教更有说服力。观众代入的是"担心亲人被骗"的视角，天然具有共情基础

## 技能文件说明

### 1. 叙事规划手法 (`director_planning_narrative.md`)

导演在 **叙事规划阶段** 使用的技法参考，定义社会教育在主题立意、叙事节奏、场景情绪设计与声音方向上的整体规划方法。

| 章节 | 内容概要 |
|---|---|
| 主题立意与叙事内核 | 真实即力量、体验式教育、关键信息视觉化、家属视角 |
| 叙事结构与节奏规划 | 60秒情感过山车、钩子→递进→冲突→揭露→醒悟→升华 |
| 分场景情绪设计 | 六大典型段落设计、现实场景化、教育与叙事平衡 |
| 声音与音乐方向 | 音乐克制、环境音真实感、静默的教育力量、关键音效标志 |
| 构图与景别叙事 | 社会教育构图逻辑、屏幕/文件特写、对峙构图、温暖收束 |
| 镜头运动与节奏 | 客观记录的固定机位、逼近真相的缓推、蒙太奇加速、稳定收束 |

### 2. 分镜表叙事手法 (`director_storyboard_table_narrative.md`)

导演在 **分镜表制作阶段** 使用的技法参考，定义社会教育在分镜表中的景别策略、运镜逻辑、时长把控、关键信息展示设计与转场手法。

| 章节 | 内容概要 |
|---|---|
| 分镜表定位 | 分镜表作为教育信息传递核心工具的定位说明 |
| 景别选择 | 悬念定场/对话中景/信息特写/高潮反转全景 |
| 运镜节奏 | 客观固定、逼近推镜、蒙太奇快切、温暖缓拉 |
| 时长把控 | 钩子时长/对话节奏/蒙太奇加速/信息停留/留白消化 |
| 关键信息展示设计 | 屏幕文字/文件/横幅/道具文字的画面呈现规范 |
| 角色表情设计 | 从固执→疑惑→震惊→醒悟的表情序列节奏 |
| 台词与留白 | 精简台词、不说教原则、关键信息用画面、留白的教育力量 |
| 转场设计 | 硬切/色彩转场/空镜过渡/信息呼应转场 |

## 使用方式

本技能包作为 ToonFlow 导演 AI 的叙事手法参考，在故事创作流程中自动加载，指导：

1. **叙事规划** — 确定教育主题、情感节奏、场景氛围与信息展示方式
2. **分镜表生成** — 根据叙事规划输出具体的景别、运镜、时长与信息视觉化方案

两个文件分工明确、逐层递进，从"为什么这样拍"到"具体怎么拍"形成完整闭环。
