# 分镜表社会教育风格约束 · 2D动漫社会教育短片 · 技法参考

---

## 一、分镜表定位

分镜表是导演将剧本转化为镜头语言的核心工具。以下仅提供2D动漫社会教育风格在分镜表层面的约束规范。

---

## 二、色彩与氛围

- **同场戏色彩统一** — 一场戏内不应出现两种以上色彩方案，除非有明确的叙事转折（如暖色调突然切换为警示高冷色调）
- **情绪色盘绑定** — 每场戏至少绑定 1 个情绪场景（如温馨日常/警示提醒/安全教育/危险场景等），并在镜头组内保持主色+辅色连续性
- **色彩对比规范** — 默认中对比；警示段落用高对比（警示红+暗夜蓝），日常段落用低对比（米白+浅黄）
- **警示色释放规则** — 警示色（红/橙）默认10%以下画面占比，仅在风险暴露/错误发生镜头可提升至30-50%，制造视觉冲击
- **色彩转场是高级手段** — 从暖色渐变到冷色 = 情绪突变。在分镜表中标注色彩变化点
- **信息层级色** — 关键信息/警示元素使用红/橙高亮，引导观众注意力

---

## 三、色块动态

- **色块变化增加画面呼吸感** — 色相变化、明度变化、大小变化。每 3-4 个镜头至少安排一个有色块变化的镜头，避免画面"死"掉
- **社会教育风格环境元素优先** — 色块动态应选用社会教育风格内的元素：色相渐变、色块位移、明度过渡、几何形状变化，禁止出现写实/3D 元素
- **警示色块闪烁** — 在危险/警示段落可使用色块明度快速变化模拟警示灯效果

---

## 四、动作节奏

- **动作要简洁** — 所有人物动作默认自然节奏。抬手、转身都应标注"简洁"或"平滑"
- **单镜头动作不超过两个** — "抬手 + 皱眉"可以，"抬手 + 转身 + 行走 + 说话"会崩
- **教育演示用符号化** — 展示知识/信息时，可用几何图标/色块动画辅助说明，避免纯对白教学
- **表情变化优先** — 社会教育短片中，角色表情变化是情感传递的核心载体

---

## 五、运镜禁忌

- **禁用复杂运镜** — 旋转、扭曲、快速切换与本风格气质冲突
- **禁用花式转场** — 旋转、扭曲、擦除等与本风格不兼容
- **默认静止** — 70% 以上镜头应为固定机位，让画面的色块和构图自己说话
- **警示段落可有限摇镜** — 仅限危险揭示、关键信息暴露时使用缓慢摇镜或推镜

---

## 六、转场规范

- **默认硬切** — 60秒教育短片信息密度高，默认使用硬切保持节奏
- **色块过渡** — 段落转场使用全屏色块过渡（如：全屏警示红→下一场景），增强情绪冲击
- **符号化转场** — 可用几何形状（感叹号/问号/箭头）放大至全屏作为转场手段

---

## 七、分镜卡片模板

| 字段 | 内容 |
|---|---|
| 镜号 | 按时间顺序编号 |
| 画面描述 | 画面内容分三段：① 角色/场景布局 ② 角色表情与动作 ③ 信息层级与色块分布 |
| 景别 | 远景/全景/中景/中近景/特写/大特写 |
| 运镜 | 固定/慢推/慢拉/摇镜（限警示段） |
| 时长 | 单位秒，60秒短片总时长控制 |
| 角色动作 | 角色做什么+表情变化 |
| 对白/旁白 | 角色台词或画外旁白 |
| 声音 | 配乐+环境音+声效 |
| 色彩氛围 | 绑定prefix情绪色盘的主色+辅色 |
| 字幕/信息卡 | 画面叠加的教育信息/警示文字 |

---

## 八、质量自检清单

| 编号 | 检查项 |
|---|---|
| Q1 | 前7秒是否建立冲突或抛出问题？ |
| Q2 | 警示色释放时机是否合理？是否制造了情绪冲击？ |
| Q3 | 信息密度是否过高或过低？（每5秒一个信息点） |
| Q4 | 固定机位占比是否≥70%？ |
| Q5 | 色块动态是否每3-4镜出现一次？ |
| Q6 | 表情变化是否清晰传达当前情绪？ |
| Q7 | 是否有古装/奇幻/写实元素混入？ |
| Q8 | 色彩方案是否在prefix情绪色盘范围内？ |
| Q9 | 转场手段是否符合规范（硬切/色块过渡/符号化转场）？ |
| Q10 | 字幕/信息卡文字是否清晰可读且不遮挡关键画面？ |
