# 全局美学基础 · 2D动漫社会教育

---
必须严格、完整遵循下方全部风格约束与全局规则，并严格按提示词模板格式生成提示词；仅输出提示词正文，不得附加任何解释、说明、注释、标题或其他额外文本。

## 一、风格基因

| 维度 | 定义 |
|---|---|
| **一级风格** | 2D动漫社会教育（2D Animation Social Education） |
| **二级风格** | 动漫风格角色 · 扁平色块场景 · 清晰线条 · 纯色填充 |
| **情感基调** | 亲和力 · 警示性 · 正向引导 |
| **质感锚词** | 清晰线条、纯色色块、表情生动、色块对比 |

> 风格介于"扁平风"与"二次元"之间：保留表情丰富度的同时用色彩确保信息传达效率。

---

## 二、全局色彩盘（风格基线，不是硬锁）

> 目标：统一审美而非限制创作。除「硬约束色」外，其余颜色默认优先使用，可在合理范围内偏移。

### 色彩使用层级

| 层级 | 约束强度 | 说明 |
|---|---|---|
| L1 硬约束 | 高 | 仅锁定角色识别核心：肤色、发色、主服底色的审美方向 |
| L2 软约束 | 中 | 场景色、配饰色、点缀色优先参考色盘，可按镜头与剧情微调 |
| L3 例外机制 | 低 | 高潮/警示/特殊场景可临时突破局部色彩，但需保留整体动漫扁平逻辑 |

|---|---|---|---|
| C1 | 警示红 | #EF4444 | 警示/危险/重点信息高亮 |
| C2 | 安全蓝 | #3B82F6 | 安全/理性/官方信息 |
| C3 | 纯净白 | #FFFFFF | 背景、留白、纯净感 |
| C4 | 深棕发 | #4A3728 | 发色、眼瞳 |
| C5 | 高级灰 | #8A8A8A | 中性色、次要元素 |
| C6 | 警示橙 | #F59E0B | 警示/注意/温暖点缀 |

| C10 | 暗夜蓝 | #1E3A5F | 夜晚、危险、严肃氛围 |

### 硬约束色（默认锁定）

| 色项 | 对应色 | 规则 |
|---|---|---|
| 肤色基准 | C3 纯净白 + C9 米白 | 默认优先，允许明度微调 |
| 发色/瞳色基准 | C4 深棕发 | 默认优先，允许深棕/深褐/纯黑偏移 |

### 软约束色（推荐优先）

> C1/C2/C6/C7/C8/C10 为推荐色域，用于服装、装饰、背景、警示信息等。可根据镜头氛围做同色相邻近调整。

### 情绪色盘（导演对齐版）

| 情绪场景 | 主色 | 辅色 | 色块对比建议 | 画面关键词 |
|---|---|---|---|---|
| 温馨日常 | C9 米白 | C3 纯净白 + C8 浅黄 | 低对比，柔和 | 生活感、温暖、平静 |
| 警示提醒 | C1 警示红 | C2 安全蓝 + C3 纯净白 | 高对比，醒目 | 警惕、危险、注意 |
| 安全教育 | C2 安全蓝 | C3 纯净白 + C7 宁静绿 | 中对比，理性 | 知识、规范、正确 |
| 正向引导 | C7 宁静绿 | C9 米白 + C2 安全蓝 | 低对比，舒适 | 希望、安全、信赖 |
| 危险场景 | C1 警示红 | C10 暗夜蓝 + C6 警示橙 | 高对比，紧张 | 危险、紧迫、警示 |
| 回忆/闪回 | C8 浅黄 | C5 高级灰 + C9 米白 | 低对比，柔和 | 怀旧、反思、回忆 |
| 错误/陷阱 | C6 警示橙 | C10 暗夜蓝 + C5 高级灰 | 高对比，压抑 | 迷失、错误、陷阱 |
| 成功/解脱 | C7 宁静绿 | C3 纯净白 + C8 浅黄 | 中对比，明亮 | 释然、正确、解脱 |

### 情绪色盘使用规则

| 编号 | 规则 |
|---|---|
| E1 | 每条提示词至少指定 1 个「情绪场景」并绑定主色+辅色组合 |
| E2 | 单镜头主色不超过 2 个，避免颜色叙事失焦 |
| E3 | 情绪切换时优先调整色相较色温，再调整饱和度 |
| E4 | 教育短片默认遵循「理性底 + 警示高亮」：理性色铺底，警示色用于关键信息高亮 |
| E5 | 若与剧情冲突，以情绪色盘优先于通用推荐色，但不得突破严禁项 |

### 色温约束

| 参数 | 值 | 说明 |
|---|---|---|
| 整体色温 | 中性 5500-6500K（推荐） | 清晰明快主基调 |
| 肤色色温 | 微暖 5800-6200K（推荐） | 米白但有生命感 |
| 对比度 | 中等偏高（建议保持） | 色块对比清晰，教育信息易读 |
| 饱和度 | 中高 70-90%（建议区间） | 动漫风格用色 |

### 容差与例外

| 项目 | 建议容差 |
|---|---|
| 色相偏移 | ±8° |
| 饱和度偏移 | ±10% |
| 明度偏移 | ±12% |

> 例外场景：警示高潮、危险转折镜头可使用更高饱和警示色；但禁止高饱和荧光色与霓虹色入镜。

## 三、全局约束规则

### 必守规则（所有技能继承）

| 编号 | 规则 |
|---|---|
| R1 | 必须包含「2D动漫 + Social Education」风格锚定词 |
| R2 | 必须声明「清晰线条 + 纯色填充 + 色块对比」 |
| R3 | 面部必须使用「动漫风格五官 + 表情生动」 |
| R4 | 轮廓线必须使用「清晰轮廓 + 线条均匀」 |
| R5 | 色彩必须声明「纯色填充 + 色块对比清晰 + 无渐变」 |
| R6 | 角色必须为现代日常着装，严禁古装/仙侠/奇幻服饰 |
| R7 | 角色表情必须保持适度丰富度，区别于纯扁平风的极简表情 |

### 严禁项（所有技能继承）

| 编号 | 严禁内容 |
|---|---|
| X1 | 严禁「3D渲染/写实渲染/照片级真实感」 |
| X2 | 严禁「阴影/渐变/复杂纹理/光影」 |
| X3 | 严禁「高饱和荧光色/霓虹色」 |
| X4 | 严禁「面部变形/比例失调/肢体异常」 |
| X5 | 严禁「复杂细节/精细纹理/写实背景」 |
| X6 | 严禁「古装/仙侠/奇幻元素」 |
| X7 | 严禁「血腥暴力/恐怖黑暗/成人向内容」 |
