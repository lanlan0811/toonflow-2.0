# 人物基础形象生成 · 2D动漫社会教育约束手册

---

## 一、基础形象原则

1. **动漫为本** — 保留二次元风格的五官表现力与表情丰富度，区别于纯扁平风的极简面容
2. **现代着装** — 全部角色穿着当代日常服装（T恤/衬衫/校服/制服等），由角色描述（身份/职业/性别/场景）决定其常规着装
3. **色块表达** — 人物造型以色块分层构建，无阴影渐变，靠色块对比表达层次
4. **四视图一致** — 轮廓/体型/发型/基础服装跨视图高度统一

---

## 二、面容约束

> 由角色描述（性别/年龄/性格/气质）驱动 AI 生成动漫风格五官，保留二次元的眼睛表现力。

### 通用要求

| 项目 | 约束 |
|---|---|
| 五官 | 动漫风格五官，由角色描述自然推导，保留大眼睛/清晰表情特征 |
| 风格底色 | 2D动漫风格、清晰线条、纯色色块、表情生动 |
| 气质 | 必须从角色描述提炼整体气质关键词，并写入提示词 |
| 表情 | 根据情绪场景选择对应表情，介于扁平风与二次元之间 |

## 六、基础服装约束

> 基础服装由角色描述（身份/职业/性别/场景）决定其最自然的常规着装，作为该角色「日常默认态」。**禁止内衣打底**。

### 着装选择原则

| 角色身份 | 默认着装方向 |
|---|---|
| 学生 | 校服 / 学院装（现代款） |
| 上班族 | 职业便装（衬衫+裤装/裙装） |
| 居家/休闲 | 休闲常服（T恤+裤装/连衣裙） |
| 运动/活泼 | 运动套装 |
| 特殊职业 | 对应身份服装（医生/警察/教师/社区工作者等） |
| 中老年 | 符合年龄的日常便装 |
| 角色描述未明确 | 简洁日常常服 |

### 着装统一规则

- 服装风格须与2D动漫风格一致（纯色色块、无渐变、无光影）
- 颜色低饱和纯色填充，无复杂花纹/装饰
- 四视图服装款式完全一致
- 基础服装是「日常默认态」，焦点仍在面容与体态
- **严禁内衣/暴露/性化打底**
- **严禁古装/仙侠/奇幻服饰**

## 七、四视图设定图规范

### 视图定义

| 位置 | 视图 | 角度 | 景别 | 要求 | 提示词 |
|---|---|---|---|---|---|
| 左一 | 人像特写 | 正面平视 | 头顶至锁骨 | 从头顶到锁骨完整展示不裁切，面部占60%+，五官清晰 | portrait closeup、face detail、head to collarbone complete |
| 左二 | 正视图 | 正面 0° | 全身立像 | 面对镜头、双臂自然、从头顶到脚底完整展示 | front view、full body head to toe |
| 右二 | 侧视图 | 右侧 90° | 全身立像 | 纯侧面轮廓清晰、从头顶到脚底完整展示 | side view、profile、full body head to toe |
| 右一 | 后视图 | 后方 180° | 全身立像 | 后脑/背部/发尾/脚部清晰、从头顶到脚底完整展示 | back view、rear view、full body head to toe |

### 画面规范

| 项目 | 约束 |
|---|---|
| 布局 | 同一画面从左至右并排四视图 |
| 背景 | 纯净中性灰 #E8E8E8 |
| 站姿 | 自然站立、双脚平行微分、双臂自然下垂或微展 |
| 全身展示 | 全身立像必须从头顶到脚底完整入画，严禁裁切头顶或脚部 |
| 特写展示 | 人像特写必须从头顶到锁骨完整入画，严禁裁切头顶 |
| 表情 | 中性微表情，符合角色气质 |
| 光线 | 无光影，纯平涂色块 |
| 一致性 | 四视图的肤色/体型/发型/面容/基础服装完全一致 |
| 画面比例 | 建议 4:1 或 3:1 |

## 九、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须为「素颜无妆」状态 |
| R2 | 必须根据角色描述声明合适的现代日常着装作为基础服装；禁止内衣打底 |
| R3 | 必须声明「无发饰、无配饰」 |
| R4 | 必须指定「纯净中性灰背景」 |
| R5 | 必须指定「四视图一致性」 |
| R6 | 全身立像必须从头顶到脚底完整展示，严禁裁切 |
| R7 | 必须指定角色身高并通过头身比换算约束全身比例（女性默认155-170cm/5-6头身，男性默认170-185cm/5.5-6.5头身） |
| R8 | 人像特写必须从头顶到锁骨完整展示，严禁裁切头顶 |
| R9 | 角色服装必须为现代日常着装，严禁古装/奇幻服饰 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 内衣/暴露/性化打底；与角色描述明显不符的着装；古装/仙侠/奇幻服饰 |
| X2 | 光影/阴影/渐变效果 |
| X3 | 3D渲染/CG质感 |
| X4 | 复杂场景背景（必须纯灰底） |
| X5 | 夸张动态姿势 |
| X6 | 全身立像裁切头顶或脚底，必须从头到脚完整入画 |
| X7 | 人像特写裁切头顶，必须从头顶到锁骨完整入画 |
| X8 | 忽略身高和头身比约束 |
