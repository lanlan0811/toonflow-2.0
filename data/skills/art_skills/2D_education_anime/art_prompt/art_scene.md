# 场景图生成 · 2D动漫社会教育约束手册

---

## 一、场景美学原则

1. **空间叙事** — 场景承载情绪与叙事功能，服务于教育信息传达
2. **层次扁平** — 前/中/后景通过色块区分，无纵深透视
3. **色块至上** — 所有场景必须通过色块表达，拒绝渐变/光影
4. **现代题材** — 全部为当代生活场景（室内/街道/办公室/学校/社区等），禁止古装/仙侠/奇幻场景
5. **信息辅助** — 场景中的色块布局应辅助信息传达，引导观众视线

## 三、室内场景

### 空间规范

| 维度 | 约束 | 提示词 |
|---|---|---|
| 风格 | 现代简约，日常居住/办公/学习空间 | 现代室内、动漫风格 |
| 材质 | 纯色色块、线条勾勒 | 扁平木质、扁平墙面、扁平布料 |
| 色调 | 中低饱和纯色色块 | 动漫暖色、动漫冷色 |
| 纵深 | 前/中/后景色块区分 | 前景{色块}、中景{色块}、后景{色块} |
| 质感 | 无质感、纯色填充 | 无纹理、动漫质感、flat texture |
| 光照 | 无光照、纯平涂色块 | 无光影、扁平光照、no lighting |
| 镜头感 | 无景深虚化、纯平面 | 无景深、扁平视角、no depth |

## 五、主视图规范

### 视图定义

> 单画面主视图，从场景中最具代表性的角度拍摄，承载空间叙事与构图重心。

| 项目 | 约束 | 提示词 |
|---|---|---|
| 视角 | 自然观察视角，构图最能体现场景主体与色块层次 | hero shot、representative angle |
| 视点高度 | 默认人眼平视高度，特殊场景可俯/仰 | eye level（默认） |
| 构图 | 主体居中或符合三分法，前/中/后景色块层次清晰 | balanced composition |

### 画面规范

| 项目 | 约束 |
|---|---|
| 布局 | 单画面（非拼图、非多视图、非分屏） |
| 人物 | **严禁出现任何人物、人影、人体轮廓** |
| 一致性 | 风格/材质/色调统一 |
| 光线 | 无光照、纯色填充、无光影逻辑 |
| 画面比例 | 默认 1:1（或按调用方设定） |
| 年代感 | 均为现代场景，无古风/模仿古风元素 |

## 七、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 场景必须有「前中后景色块层次」 |
| R2 | 室外必须为「扁平远景」无空气透视 |
| R3 | 场景图必须为「单画面主视图」，不得拼接多视图/分屏/网格 |
| R4 | 构图须能代表场景主体并展示前/中/后景色块层次 |
| R5 | 场景图中**严禁出现任何人物** |
| R6 | 必须指定「动漫风格」关键词（2D anime style、flat illustration） |
| R7 | 必须指定「无光影无渐变」 |
| R8 | 材质必须为纯色填充，禁止复杂纹理 |
| R9 | 场景必须为现代题材，禁止古装/仙侠/奇幻场景 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 背景纯白/纯黑/无场景 |
| X2 | 极端天候（暴风雨/雷电/暴雪，除非剧情需要） |
| X3 | 场景无层次/无色块区分 |
| X4 | 植被/天候与季节矛盾 |
| X5 | 出现任何人物、人影、人体剪影或人体轮廓 |
| X6 | 画面被拼接成多视图/网格/分屏布局 |
| X7 | 3D 渲染/CG动画/游戏引擎质感 |
| X8 | 材质过于复杂、色块区分不清 |
| X9 | 添加光影/阴影/渐变/立体感效果 |
| X10 | 古装/仙侠/奇幻/历史场景元素 |
