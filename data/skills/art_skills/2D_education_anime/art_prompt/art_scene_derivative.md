## 一、不同时段变体

> 在基础场景主视图基础上，生成同一场景在不同时段下的色调/光线/氛围变体。所有衍生必须严格保持与基础场景一致的空间布局/建筑结构/家具陈设。

### 时段变体类型

| 时段 | 色调特征 | 色块方案 | 氛围词 |
|---|---|---|---|
| 清晨 | 浅蓝+暖黄渐变 | 低对比，冷转暖 | 清新、苏醒 |
| 上午 | 纯净白+浅蓝 | 中对比，明亮 | 活力、高效 |
| 正午 | 纯白+亮黄 | 高对比，明亮 | 强烈、专注 |
| 下午 | 米白+暖橙 | 中对比，温暖 | 悠闲、温和 |
| 黄昏 | 橙红+紫色 | 高对比，暖转冷 | 温馨、静谧 |
| 夜晚 | 暗蓝+暖黄点缀 | 低对比，冷为主 | 宁静、神秘 |
| 深夜 | 深蓝+黑色 | 极低对比，冷调 | 寂静、沉睡 |

## 四、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有场景衍生必须保持与基础场景一致的空间布局/结构 |
| R2 | 时段/天气/情绪变化必须通过色调和色块方案体现 |
| R3 | 必须声明「画面中无任何人物」 |
| R4 | 必须包含动漫风格锚定词 |
| R5 | 必须声明「无光影无渐变」 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 改变场景的空间布局和主要结构 |
| X2 | 画面中出现人物/人影/人体轮廓 |
| X3 | 古装/仙侠/奇幻场景元素 |
| X4 | 渐变/阴影/光影效果 |
| X5 | 3D渲染/CG质感 |
| X6 | 极端天候（除非剧情明确需要） |
