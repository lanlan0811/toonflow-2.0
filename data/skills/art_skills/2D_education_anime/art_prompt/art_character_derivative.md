## 一、不同情绪状态变体

> 在基础角色设定图基础上，生成同一角色在不同情绪场景下的面容/表情/体态变体。所有衍生必须严格保持与基础设定图一致的发型/发色/肤色/服装/体型。

### 情绪变体类型

| 情绪 | 面容特征 | 眼神 | 体态 | 色块氛围 |
|---|---|---|---|---|
| 警惕/怀疑 | 眉头微皱 | 眼睛微眯，目光锐利 | 身体微微后倾 | 冷色+红色高亮 |
| 惊讶/震惊 | 眉毛上扬 | 瞳孔放大，眼神集中 | 身体定格 | 白色+橙色高亮 |
| 恐惧/害怕 | 眉头紧锁 | 眼睛放大，目光不安 | 身体退缩 | 暗色+红色低饱和 |
| 安心/信任 | 嘴角微笑 | 眼神柔和，目光温暖 | 身体放松前倾 | 暖色+绿色柔和 |
| 愤怒/不满 | 眉头紧锁 | 眼睛变小，目光尖锐 | 身体前倾 | 冷色+红色强对比 |
| 悲伤/失落 | 嘴角下弯 | 眼神暗淡，目光无神 | 低头含胸 | 冷色+灰色调 |
| 困惑/迷茫 | 眉头微蹙 | 目光散乱，眼神犹疑 | 身体微倾 | 灰色+冷色混合 |
| 后悔/自责 | 眉头深锁 | 眼睛向下，目光躲避 | 低头垂肩 | 冷色+暗色调 |
| 希望/欣慰 | 嘴角上扬 | 眼神明亮，目光柔和 | 身体挺直 | 暖色+绿色明亮 |
| 坚定/醒悟 | 表情决然 | 眼神坚定，目光集中 | 身体直立 | 蓝+绿明亮对比 |

## 二、不同服装变体

> 在基础角色设定图基础上，生成同一角色在不同场景/季节/角色身份下的服装变体。所有衍生必须严格保持与基础设定图一致的面容/发型/发色/肤色/体型。

### 服装变体类型

| 变体类型 | 适用场景 | 服装方向 |
|---|---|---|
| 校服变体 | 学生角色上学/放学 | 现代校服（衬衫+外套+裤子/裙子） |
| 职业变体 | 上班族通勤/办公 | 衬衫+西裤/半身裙+西装外套 |
| 休闲变体 | 日常生活/居家 | T恤+牛仔裤/休闲裤/连衣裙 |
| 运动变体 | 运动/户外 | 运动服+运动鞋 |
| 正装变体 | 正式场合 | 西装/礼服 |
| 睡衣变体 | 居家/休息 | 睡衣/居家服 |
| 季节变体 | 春夏秋冬 | 短袖/长袖/外套/羽绒服 |
| 制服变体 | 特殊职业 | 医生/警察/消防员/志愿者制服 |

### 提示词模板（服装变体）

```
图中不要有任何文字，

Based on the reference image of [角色名称], maintain consistent: face features, hairstyle, skin color, body proportion. Generate a new costume variant: {服装描述}.
```

## 三、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有衍生变体必须与基础设定图保持面容/发型/发色/肤色/体型一致 |
| R2 | 情绪变体必须通过面部表情和眼神变化体现 |
| R3 | 服装变体必须为现代日常着装 |
| R4 | 必须声明「纯净中性灰背景」 |
| R5 | 必须声明「无阴影无渐变」 |
| R6 | 必须包含动漫风格锚定词 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 改变基础角色的面容/发型/发色/肤色/体型特征 |
| X2 | 古装/仙侠/奇幻服饰变体 |
| X3 | 暴露/性化服装变体 |
| X4 | 渐变/阴影/光影效果 |
| X5 | 3D渲染/CG质感 |
| X6 | 复杂场景背景（必须纯灰底） |
