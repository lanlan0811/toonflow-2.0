# 道具衍生图像 · 2D动漫社会教育约束手册

---

## 一、不同使用状态变体

> 在基础道具设定图基础上，生成同一道具在不同使用状态或场景下的形态变体。所有衍生必须严格保持与基础设定图一致的道具主体造型/色彩/结构。

### 使用状态变体类型

| 状态变体 | 视觉特征 | 适用道具类型 | 说明 |
|---|---|---|---|
| 关闭/合拢 | 完整闭合形态 | 手机/电脑/书本/雨伞/背包 | 静止状态，未激活 |
| 打开/展开 | 展开/开启形态 | 电脑/书本/雨伞/折叠椅 | 使用中状态 |
| 激活/亮屏 | 屏幕亮起/指示灯 | 手机/电脑/电子设备 | 有信息显示 |
| 装满/使用中 | 内容物可见 | 水杯/饭盒/背包/购物袋 | 承载状态 |
| 空置/未使用 | 空载形态 | 容器/包装/储物 | 闲置状态 |
| 破损/异常 | 破损/故障/裂痕 | 警示类道具 | 异常警示状态 |
| 组合/配套 | 道具+配件组合 | 充电器+手机/笔+本 | 组合使用状态 |
| 包装/未开封 | 带包装形态 | 食品/药品/快递盒 | 未开封状态 |

### 提示词模板（使用状态变体）

```
动漫现代道具{使用状态}变体图，
2D动漫风格，social education，anime style，clean lines，solid colors，
与基础道具设定图保持完全一致的主体造型、主体色彩、主体结构，
{道具类型}，{材质描述}，{使用状态描述}，{状态视觉特征}，
纯道具静物展示，道具独立陈列，无人持有，无人握持，
纯净中性灰背景，无光影，无渐变，
线条清晰，色块分明，
现代设计，无古风元素
图中不要有任何文字，
画面中不能出现任何人物、手部、手指、肢体，道具不可处于被握持或佩戴状态
```

## 二、不同场景变体

> 在基础道具设定图基础上，生成同一道具在不同使用环境或场景色调下的变体。道具主体造型不变，通过色彩调整匹配场景氛围。

### 场景变体类型

| 场景色调 | 道具色彩调整 | 适用场景 |
|---|---|---|
| 温馨日常 | 暖色调为主 | 居家/休闲/日常环境 |
| 理性办公 | 冷色调为主 | 办公室/学校/正式场合 |
| 警示氛围 | 红色/橙色高亮 | 危险/警示/注意场景 |
| 安全正向 | 绿色/蓝色基调 | 安全/正确/正向场景 |
| 回忆怀旧 | 低饱和暖色调 | 回忆/反思/闪回 |
| 危险紧急 | 暗色调+红色高亮 | 危险/紧急/错误场景 |

### 提示词模板（场景变体）

```
动漫现代道具{场景色调}变体图，
2D动漫风格，social education，anime style，clean lines，solid colors，
与基础道具设定图保持完全一致的主体造型和结构，
{道具类型}，{材质描述}，{场景色调}色彩适配，
纯道具静物展示，道具独立陈列，无人持有，
纯净中性灰背景，无光影，无渐变，
线条清晰，色块分明，
现代设计，无古风元素
图中不要有任何文字，
画面中不能出现任何人物、手部、手指、肢体
```

## 三、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有衍生变体必须与基础道具设定图保持主体造型/结构/色彩一致 |
| R2 | 使用状态变体必须只改变功能状态，不改变道具本体造型 |
| R3 | 场景变体通过色彩调整匹配场景氛围 |
| R4 | 必须声明「纯净中性灰背景」 |
| R5 | 必须声明「无阴影无渐变」 |
| R6 | 纯道具静物展示，无人持有 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 改变基础道具的主体造型和结构 |
| X2 | 道具与人物同画面 |
| X3 | 出现任何人物形象或肢体 |
| X4 | 古装/仙侠/奇幻道具元素 |
| X5 | 渐变/阴影/光影效果 |
| X6 | 3D渲染/CG质感 |
