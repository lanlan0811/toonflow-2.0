## 一、视频生成提示词规范

> 视频生成提示词在分镜提示词基础上，增加时序控制、动作连贯性、镜头运动描述等视频专属要素。

### 基础结构

视频生成提示词 = 风格锚定 + 场景描述 + 角色动作 + 镜头运动 + 时序控制 + 氛围绑定 + 画质锁定 + 负向词

### 风格锚定（所有视频生成提示词必须包含）

```
2D动漫风格，social education，anime style，clean lines，solid colors，
纯色填充，无纹理无渐变，色块对比清晰，表情生动，
现代教育氛围，动漫美学，清晰信息传达
```

## 二、时序控制规范

### 时间单位映射

| 画面术语 | 建议时长 | 说明 |
|---|---|---|
| 快速闪现 | 0.5-1秒 | 警示信息/关键画面闪回 |
| 短暂停留 | 1-2秒 | 角色表情变化/动作反应 |
| 正常节奏 | 2-4秒 | 日常对话/场景建立 |
| 稍长停留 | 4-6秒 | 教育信息展示/情绪渲染 |
| 缓慢过渡 | 6-8秒 | 段落收尾/情绪升华 |

### 时序提示词

```
{temporal duration}，[slow/fast] motion，
{role action} with natural timing，
表情变化在[temporal duration]内完成
```

### 60秒短片节奏模板

| 段落 | 时间 | 节奏 | 内容 |
|---|---|---|---|
| 开场 | 0-7s | 快速 | 建立冲突/抛出问题 |
| 铺垫 | 7-15s | 中速 | 角色状态/日常情景 |
| 发展 | 15-35s | 渐快 | 问题恶化/陷阱暴露 |
| 高潮 | 35-50s | 快速 | 警示/危险/错误揭示 |
| 解决 | 50-60s | 中速→舒缓 | 正向引导/正确做法 |

## 三、镜头运动规范

| 运动类型 | 适用场景 | 提示词 |
|---|---|---|
| 固定镜头 | 默认，70%+镜头 | static shot，fixed camera |
| 慢推 | 揭示关键信息/情感升华 | slow dolly in |
| 慢拉 | 段落结束/情绪释放 | slow dolly out |
| 缓慢摇镜 | 展示场景全貌/警示揭示 | slow pan left/right |
| 快速推镜 | 警示信息/关键细节曝光 | fast push in |
| 色块转场 | 段落切换/情绪转折 | color block transition |

### 运镜禁忌

- 禁用旋转镜头（camera rotation）
- 禁用快速切换（rapid cut）
- 禁用晃动镜头（shaky camera）
- 禁用花式转场（wipes/zoom transitions）

## 四、提示词模板

### 模板A：视频分镜生成（单个镜头）

```
2D动漫风格，social education，anime style，clean lines，solid colors，
纯色填充，无纹理无渐变，色块对比清晰，表情生动，
{景别}，{镜头运动}，
{场景描述}，
角色{角色动作}，表情{情绪状态}，
{色彩氛围}，{主色+辅色}，
现代教育氛围，动漫美学，清晰信息传达，亲和力视觉，
{时序控制}，
高清画质，线条清晰，色彩纯净，画面无杂色无噪点，无阴影无渐变
```

### 模板B：短视频序列生成（多镜头连贯）

```
2D动漫风格短视频序列，
social education short film，anime style，clean lines，solid colors，
storyboard sequence：[镜1描述（{时长}）→ 镜2描述（{时长}）→ 镜3描述（{时长}）]，
角色保持一致性：相同发型、发色、肤色、服装，
场景保持一致性：相同空间布局、色块方案，
纯色填充，无纹理无渐变，色块对比清晰，
{整体情绪弧线：如"从温馨日常到警示警惕再到正向引导"}，
{色彩方案序列：如"方案A→方案B→方案D"}，
现代教育氛围，动漫美学，清晰信息传达，
画面中无字幕/无UI层文字
```

## 五、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| V1 | 必须包含「2D动漫风格 + Social Education」风格锚定 |
| V2 | 必须声明「无阴影无渐变 + 纯色填充」 |
| V3 | 必须指定镜头运动类型（默认固定镜头） |
| V4 | 必须指定时序/时长控制 |
| V5 | 多镜头序列必须保持角色和场景一致性 |
| V6 | 必须声明「画面中无字幕/无UI层文字」 |
| V7 | 必须绑定情绪色盘的主色+辅色 |

### 严禁

| 编号 | 严禁 |
|---|---|
| VX1 | 旋转/晃动/快速切换等复杂运镜 |
| VX2 | 花式转场效果 |
| VX3 | 3D渲染/CG动画质感 |
| VX4 | 写实渲染/照片级真实感 |
| VX5 | 古装/仙侠/奇幻元素 |
| VX6 | 画外叠加字幕/水印/标题卡等UI层文字 |
| VX7 | 高饱和荧光色/霓虹色 |
