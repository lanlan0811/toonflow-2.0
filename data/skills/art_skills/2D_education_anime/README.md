# 2D动漫教育短片 · 风格说明

本风格专为"2D动漫社会教育短片"题材打造，所有美术提示词、规范和生成内容均严格限定于：

- **2D动漫社会教育风格**：以2D动漫美学（2D Animation）为核心，融合扁平化设计（Flat Design）的简洁色块表达，专为社会教育/公益宣传题材打造。
- **亲和力视觉语言**：用动漫的亲和力降低说教感，用简洁清晰的视觉语言传递核心教育信息。角色表情丰富生动，色彩明快而不刺眼。
- **信息传达优先**：在保留二次元风格表情丰富度的同时，采用色块化表达确保信息传达效率。介于"扁平风"与"二次元"之间，取两者之长。

## 风格核心定义

| 维度 | 定义 |

| 一级风格 | 2D动漫社会教育（2D Animation Social Education） |

| 情感基调 | 亲和力、警示性、教育意义、正向引导 |

## 视觉风格定位

### 角色设计

- **动漫风格**：保留二次元的眼睛表现力、表情丰富度，角色辨识度高
- **轮廓清晰**：色块分明，线条简洁但不失生动
- **服装写实化**：角色穿着现代日常服装（T恤/衬衫/校服/制服等），符合社会教育题材的真实感需求
- **无古装元素**：全部角色均为现代着装，贴近观众日常生活

### 场景色彩

- **色彩明快**：基于现代生活场景，色调贴近真实但经过动漫化提纯
- **色块叙事**：通过色块对比传递情绪，暖色=警示/温暖，冷色=冷静/危险
- **信息层级**：关键信息通过色彩高亮引导观众注意力

### 道具规范

- **现代道具**：全部为当代生活常见物品（手机/电脑/文件/背包等）
- **功能清晰**：道具用途一目了然，通过色块简化和线条勾勒确保辨识度
- **符号化表达**：关键道具可用符号化方式呈现，增强信息传递效率

## 严禁内容

- 写实渲染、3D渲染、照片级真实感
- 阴影、渐变、复杂光影纹理
- 过度复杂细节、精细写实背景
- 黑暗暴力、血腥恐怖、成人向内容
- 古装/仙侠/奇幻元素（不符合社会教育现实题材）
- 画面出现文字/字幕/水印等UI层信息
