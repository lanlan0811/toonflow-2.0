# 分镜表社会教育约束 · 真人实拍社会教育 · 技法参考

---

## 一、分镜表定位

分镜表是导演将剧本转化为镜头语言的核心工具。本约束面向 Seedance 2.0 视频生成，所有光影、动作、空间描述均采用模型可执行的具象语言。不做抽象的情感概括，不写模型看不懂的渲染参数。

---

## 二、Seedance 2.0 描述铁律

> 社会教育分镜表中所有描述必须遵守以下翻译原则——将"导演意图"翻译为"AI可执行的物理指令"。

| 禁止的抽象表达 | Seedance 2.0 具象替换 |
|---|---|
| 他很后悔 | 眉眼低垂，目光低垂看着桌面，嘴角自然向下，右手无意识地摩挲左手拇指，呼吸缓慢而沉重 |
| 审讯室很压抑 | 冷白荧光灯管顶光（约5500K）从正上方照明，房间无暖色光源，白墙反射冷白光线，人物眼窝和鼻下形成深阴影 |
| 受害者很害怕 | 身体微缩在椅子上，双肩内收，双手交握放在腿上，目光不敢直视对方，快速眨眼 |
| 他拿出手机看短信 | 右手从裤袋取出手机，拇指由下向上滑动解锁屏幕，目光聚焦在屏幕上半部分，眉头逐渐皱起——全程约3秒 |
| 家庭氛围很温馨 | 暖色窗光从画面左侧窗户斜入（约3500K），在地面投下窗框的暖金色光斑，茶几上有冒着热气的水杯，沙发上的毛毯随意搭着 |
| 他在被跟踪 | 一个人在深夜街道上行走，身后约15米处另一个黑色轮廓跟随，两人之间始终保持距离，当被跟踪者回头时，跟踪者闪入楼栋阴影中 |

## 三、光影与氛围——Seedance 2.0 物理光源描述

### 3.1 同场戏光影统一

一场戏内不应出现两种以上核心光影方案，除非有叙事驱动的光源变化（如有人打开台灯、从室外走进派出所、从审讯开始到真相大白时的光线变化）。光源变化必须在分镜表中标注触发事件。

### 3.2 光源语句法（每个镜头必须包含）
[光源] 主光：{光源类型}，{方向}，{色温值K}，{软硬}
[光源] 辅光/环境光：{光源类型}，{方向}，{色温值K}
[光比] 约1:{X}

### 3.3 情绪 → 光源矩阵

| 情绪 | 光源配方（Seedance 2.0 可直接填入） | 视觉关键词 |
|---|---|---|
| 日常安全 | 大面积窗光自然漫射（4500-5500K），窗帘柔化或打开。光比约1:1.5低反差 | 通透、自然、日常 |
| 执法严肃 | 荧光灯管冷白顶光（5000-6500K），白墙反射补光，无暖色调。光比约1:4 | 冷白均匀、无所遁形 |
| 温情回忆 | 暖色窗光/台灯侧入（2800-3500K），暗部保留物体轮廓。光比约1:3 | 暖调包裹、怀旧 |
| 悬疑逼近 | 单光源侧打（冷白5000K或暖黄2800K），房间大部沉入暗部。光比约1:8 | 不安、危机靠近 |
| 希望救赎 | 逆光/窗光斜入（3500-4500K暖白），光感通透，面部有柔和受光面。光比约1:2.5 | 暖调明亮、新生 |
| 深夜反思 | 单一暖色台灯或窗外街灯（2800-3200K）孤岛式照明。光比约1:6 | 私密、脆弱 |
| 高压对峙 | 顶光/侧硬光冷白（5000-6500K），高反差，面部深影。光比约1:10 | 紧张、对抗 |

### 3.4 冷暖色调与叙事阶段

- **冷光主导**（5000K+）：执法审讯、严肃对话、真相揭示、高压对峙、悔恨反思
- **暖光主导**（2800-3500K）：温情回忆、安全日常、家庭场景、希望救赎
- **冷暖共存**：过渡时刻（从家庭暖光来到派出所冷光）、被骗过程中的不安（暖环境+冷暗示）
- **光源变化 = 叙事信号**：从家庭暖色窗光到派出所冷白荧光灯 = 从安全到警示的切换；从审讯室冷白到走出门外接触到暖色夕阳 = 救赎与新生

### 3.5 Seedance 2.0 光影适配要点

- 色温数字帮助模型校准白平衡倾向：`色温约5500K荧光灯管` 优于 `冷光`
- 光比数字帮助模型建立明暗意识：`光比约1:6` 优于 `暗部较深`
- 光源必须有明确来源：`从上方荧光灯管照射` 优于 `顶光`
- 环境光的反射路径描述：`白墙反射冷白光线形成环境补光` 优于 `有环境光`

## 四、环境动态——让画面呼吸

### 4.1 动态密度

每 3-4 个镜头至少安排一个有环境动态的镜头。静态对话戏也不例外——至少有一个镜头里日光灯管的微光闪烁、窗帘被微风吹动、桌面文件被空调风吹起一角、窗外树叶在动。

### 4.2 社会教育环境动态元素（Seedance 2.0 可执行）

| 场景 | 可描述的环境动态 |
|---|---|
| 家庭室内 | 窗帘被微风轻鼓后回落（约2秒周期）、水杯口热气缓缓上升、窗外车流灯光偶尔扫过天花板、时钟秒针走动 |
| 派出所/询问室 | 日光灯管的微弱闪烁（老旧灯管）、桌面文件的纸张边缘被空调风吹动、走廊远处偶尔传来脚步声、空调出风口飘带微动 |
| 社区街道 | 行道树叶簌簌晃动、远处行人走过、路边积水被车轮碾过激起涟漪、公告栏上的海报边角被风吹起 |
| 反诈中心 | 大屏幕上的实时数据图表跳动刷新、多台显示器屏幕光变化、工位上的电话指示灯闪烁 |
| 非法聚会点 | 投影幕布上的画面变动、蜡烛火苗摇晃、窗帘被外面风吹动一个小角、地面烟头被脚踢动 |
| 学校/教室 | 窗外树叶光影在课桌上移动、粉笔灰在阳光下飘散、风扇缓慢转动 |

### 4.3 Seedance 2.0 环境动态描述规范

- 动态必须有具体轨迹和速度：`窗帘被风吹起约15cm后缓慢回落，约2秒一个周期` 优于 `窗帘在动`
- 光源动态与空间光源逻辑一致：`云朵遮住太阳时，室内窗光面积缩小约40%，持续约3秒后恢复`
- 禁止无来源的动态：没有风吹=窗帘不动。室内没有开窗=没有风。审讯室窗户是封闭的=窗帘不动

## 五、人物动作节奏——Seedance 2.0 物理逻辑具象化

### 5.1 动作描述铁律

所有人物动作必须描述：**轨迹 + 速度/时长 + 身体部位协调 + 对周围物体的影响**。

### 5.2 社会教育动作具象库

| 动作 | Seedance 2.0 可执行描述 |
|---|---|
| 交出手机 | 右手从裤袋取出手机（约2秒），拇指解锁屏幕，手机屏幕朝上放在桌面上向警察方向推约10cm，手指松开——全程约4秒 |
| 看诈骗短信 | 右手从桌面拿起手机，拇指点击屏幕唤醒，目光从屏幕顶端向下逐行浏览，约2秒后眉头逐渐皱起，嘴巴微张，左手不自觉握拳 |
| 坐下 | 身体前倾微屈膝，臀部接触金属椅面时发出轻微碰撞声，背部不是靠向椅背而是微微前倾——全程约1.5秒 |
| 低头沉默 | 从坐直状态缓慢低头约30度，目光从前方移至桌面，双手从桌面移至腿上或交握——全程约2秒，之后保持静止 |
| 签字（悔过书） | 右手拿起桌面上的签字笔，拇指和食指持笔，笔尖接触纸张，从左向右缓慢书写约5个字后停顿约1秒，继续书写——全程约8秒 |
| 被带走 | 两名穿制服警察分别站在左右，各用一只手握住人物上臂中段，人物从椅子上起身（约1秒），身体转向门口方向（约1秒），右脚先迈出跨过门槛——全程约4秒 |
| 敲门 | 右手握拳，指关节第二骨节轻敲木门三下（每次间隔约0.5秒），停顿约1秒后，身体微微前倾，耳朵朝向门——全程约3秒 |
| 查看证据/照片 | 双手从桌面拿起一张照片，拇指和食指夹住照片边缘，举到眼前约30cm处，目光快速扫射后又回到照片中央的某个位置——全程约4秒 |
| 掩面哭泣 | 头和肩膀同时微缩，双手抬起到面部，手掌覆盖双眼和上半面部，手指微微张开，肩膀开始小幅抖动——全程约2秒到位，之后持续 |
| 起身离开 | 双手撑住桌面，身体重心前移，从椅子上站起（约1.5秒），转身面向门口方向，拿起椅背上的外套，右脚先迈出——全程约5秒 |

### 5.3 动作节奏与叙事场景

- **日常叙事/文戏**：动作沉稳克制，每个微动作标注时长和轨迹。节奏舒缓——不是慢，是"真实的节奏"
- **情绪波动场景**：动作幅度和速度略微增加。人物可能在对话中不自觉地加快手指敲桌面的频率、呼吸带动肩膀起伏更明显、说话时手部小动作增多
- **冲突/抓捕场景**：动作干脆利落，但仍有物理轨迹。抓捕动作须具体到"右手握住对方右臂肘部上方，左腿向前半步挡在对方脚前"
- **悔恨/反思场景**：动作缓慢且有停顿。一个动作中间可能有2-3次停顿——端起水杯→停住→放下，水没有喝

### 5.4 服饰动态

社会教育角色服饰以日常便服为主，服饰动态不夸张但仍是画面真实感的来源：`夹克面料随肩膀内收而产生褶皱`、`衬衫袖口在签字时自然上移露出小臂`、`低头时长发从肩后滑落到脸侧`、`外套下摆在坐下时被压在身下后又扯出`

## 六、空间逻辑具象化——Seedance 2.0 空间坐标系统

### 6.1 每镜头必须声明的空间信息
水平位置：画面左/中/右，或画面左侧三分之一处
纵深位置：距镜头{数值}米，前景/中景/背景
人物与场景的关系：距{固定物}{数值}米
（若双人以上）人物A与B的相对距离和朝向

### 6.2 站位衔接示例
【片段A尾部】
A坐在派出所询问室的金属椅上（画面中央，距镜头约2.5米，距询问桌约0.3米）。
双手交握放在桌面，目光低垂看着桌面某点。
冷白荧光灯顶光从上方照明，桌面区域明亮，人物脸部眼窝有阴影。

【片段B开头】
A仍坐在询问椅上（位置未变，距镜头约2.5米）。
桌面上的双手已松开——右手放在桌面，左手收回放在腿上。
A缓缓抬起头约15度，看向画面左侧门口的警察方向。
荧光灯冷白光线从上方照亮抬起的脸——眼窝和鼻下的阴影角度不变。

### 6.3 空间变化必须具象

| 抽象（禁止） | Seedance 2.0 具象 |
|---|---|
| 他被带进审讯室 | 两名警察陪同A从走廊（画面左侧）走向审讯室门口，警察B推开金属门（门向内打开约90度），A右脚跨过门槛进入室内——从走廊荧光灯到室内荧光灯，色温和亮度一致 |
| 她走进派出所 | A从派出所大门外（室外天光约5500K）推开玻璃门，右脚跨入大厅（室内荧光灯约5000K），光线从日光切换为室内冷白，脚步声从室外空旷变为室内回声 |
| 从家到派出所 | 【家】A坐在客厅沙发上，暖色窗光（约3500K）照在脸上——【场景转】A坐在询问室金属椅上，冷白顶光（约5500K）从上方照明 |

## 七、运镜规范——Seedance 2.0 摄影机运动

### 7.1 允许的运镜

| 运镜 | Seedance 2.0 描述 | 适用场景 |
|---|---|---|
| 固定 | 摄影机固定不动，画面静止 | 对话、询问、审讯、空镜留白——社会教育的基础镜头语言 |
| 手持微晃 | 摄影机有轻微不规则晃动（幅度约±1.5cm），模拟手持拍摄的呼吸感 | 情绪波动、冲突时刻、慌乱/奔跑、纪实跟随 |
| 稳定器流动 | 摄影机平滑匀速移动，无晃动 | 空间展示（从外到内）、人物入场、过渡镜头 |
| 缓推 | 摄影机向主体缓慢推进，推进速率约每秒0.2-0.3米 | 逼近真相、关键信息揭示、情绪爆发的前奏 |
| 缓拉 | 摄影机向后缓慢拉远，拉远速率约每秒0.3米 | 揭示环境全貌、结尾收束、教育信息的沉淀 |
| 跟拍 | 摄影机与人物保持约1.5-2米距离同步移动 | 纪实跟随、走访、被带走的过程 |
| 摇镜 | 摄影机原地水平/垂直旋转 | 视线转移、空间关系交代、从人物到环境/证据 |

### 7.2 禁用运镜

- 无叙事目的的快速甩镜、急推急拉（推拉速率超过每秒0.8米）
- 超过2秒的剧烈手持晃动（除非主观受击/被推搡/奔跑叙事）
- 无逻辑花式转场——划屏、旋转、百叶窗、翻页等特效转场
- 摄影机无理由的360度旋转
- 模仿"偷拍"视角的极端晃动（社会教育不需要虚假的纪实伪装）

### 7.3 社会教育运镜哲学

- **固定机位是第一选择**——让观众看到真实的人在真实空间里面对真实的后果
- 手持微晃在情绪段落使用——但晃动幅度不超过纪实风格的常规范围
- 运动镜头的起幅和落幅必须平稳，运动过程匀速——禁止突然加速或急停
- 缓推在社会教育中具有强大的叙事力量：从全景缓缓推到近景，观众的注意力从"看环境"切换到"看人"，情感连接在这个过程中建立

## 八、转场规范

### 8.1 允许的转场方式

| 转场 | 视觉执行 | 叙事功能 |
|---|---|---|
| 硬切 | 直接切换 | 同场戏内镜头切换（默认），不同空间的对比切换 |
| 光影匹配转场 | 两个场景在同一光影逻辑下切换 | 时间流逝、平行叙事。例：家庭暖光→回忆中的暖光；派出所冷光→另一个审讯室冷光 |
| 空间匹配转场 | 两个空间在构图或元素上呼应 | 场景跳转。例：家中门关上→派出所门打开；手机屏幕上的诈骗短信→反诈中心的大屏幕 |
| 空镜过渡 | 插入场景空镜（3-5秒） | 情绪缓冲、章节划分、时间流逝暗示——社会教育中最常用的转场方式 |
| 焦点转场 | 前镜焦点从主体移至背景，后镜从模糊背景中逐渐聚焦主体 | 空间切换、注意力的转移、从"情绪"到"证据"的过渡 |

### 8.2 禁止的转场

- 纯视觉特效转场（翻页、划屏、百叶窗、马赛克、闪光白屏）
- 无叙事逻辑的旋转/缩放转场
- 同一场戏内使用超过两种转场方式

## 九、音画同步规划（Seedance 2.0 专属）

### 9.1 环境音标注规范

每场戏标注 1-2 个核心环境音，写在分镜表的环境音列：

| 场景 | 建议环境音 |
|---|---|
| 家庭日间 | 时钟走动 / 窗外远处车流 / 冰箱低频嗡鸣 |
| 派出所询问室 | 荧光灯管微弱嗡鸣 / 远处走廊脚步声 / 空调风噪 |
| 社区街道 | 车流胎噪 / 远处人声 / 行道树叶簌簌 |
| 反诈中心 | 键盘敲击 / 电话铃声交织 / 大屏幕数据刷新音 |
| 深夜独处 | 冰箱嗡鸣 / 窗外偶尔的车声 / 自己的呼吸声 |
| 非法聚会点 | 投影机风扇声 / 低语声 / 蜡烛火苗声 |
| 法庭 | 文件翻阅 / 法官锤轻敲 / 庄重的静默 |

### 9.2 音画同步标注

在分镜表中对关键音画同步点做标注：

## 十一、空镜使用规范

### 11.1 空镜不是"没东西拍"

空镜是情绪容器。每个空镜必须有叙事目的和具体的视觉内容：

| 空镜类型 | 叙事目的 | 示例 |
|---|---|---|
| 场景建立 | 新空间首次出现——让观众看清楚这是什么地方 | 询问室全景：金属桌椅、白墙、荧光灯管、冷白光线 |
| 情绪缓冲 | 高情绪段落后的喘息 | 窗外的雨滴沿玻璃滑落，速度约每秒2cm——让观众消化刚才的警示信息 |
| 时间流逝 | 暗示时间过去了 | 同一扇窗外的天空从午后湛蓝渐变至蓝调深紫 |
| 隐喻留白 | 以物代情 | 桌上一杯没喝完的水、一张反诈宣传单摊开、一支笔和一份空白的悔过书 |
| 转场衔接 | 前后两个空间的自然过渡 | 楼道里的日光灯管——前一场的家庭，下一场的派出所 |

### 11.2 空镜 Seedance 2.0 描述规范

空镜也必须遵循光源+质感+动态的描述铁律：
派出所询问室全景空镜——冷白荧光灯管顶光（约5500K）均匀照亮整个空间，白墙有细微纹理和磨损痕迹，
金属审讯桌表面有指纹和微划痕反射冷光，两张金属询问椅相对摆放，
墙角监控摄像头的指示灯发出微小红点，墙面挂有"依法办案"的标语牌。
画面固定，时长约4秒，只有荧光灯管的微弱嗡鸣声。

## 十二、分镜表质量自检清单

每场戏的分镜表完成后，导演逐项自检：

| 检查项 | 通过标准 |
|---|---|
| 光源可追溯 | 每个镜头能回答"光从哪儿来、什么色温" |
| 动作可执行 | 每个动作有轨迹、时长、身体部位 |
| 空间可定位 | 每个镜头标注了人物的水平和纵深位置 |
| 站位移接 | 相邻镜头的同一个人位置/姿态可衔接 |
| 环境有动态 | 每3-4个镜头至少1个有环境动态 |

| CG术语清零 | 无PBR/SSR/AO/体积光/次世代等CG词汇 |
| @reference完整 | 角色/场景/核心道具都标注了锚定引用 |
| 社会教育合规 | 警示内容不美化犯罪/邪教，不制造过度恐怖，留有希望出口 |
