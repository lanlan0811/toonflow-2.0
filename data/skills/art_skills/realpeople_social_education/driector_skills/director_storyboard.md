# 分镜提示词 · 真人实拍社会教育 · 风格专属技法

---

## 适用范围

本 Skill 专用于**真人实拍社会教育**风格的分镜提示词生成，适配反邪教警示教育、反诈骗宣传、公共安全科普、公益道德传播、法制教育等真人实拍社会教育题材的分镜创作。

## 一、摄影机前的情绪——面孔、眼神、微表情

> 社会教育短片的情绪不来自"特效"或"渲染参数"，而来自真人演员在摄影机前的微表情。以下映射面向摄影机捕捉真实人物的方式——社会教育题材的情绪有其鲜明的警示叙事特征。

| 情绪输入 | 面容描述 | 眼神描述 | 微表情补充 |
|---|---|---|---|
| 恐惧/惊慌 | 面部肌肉微绷，眉毛上抬，额头出现横纹 | 瞳孔微扩，目光不定，快速扫视后聚焦在某个点上 | 嘴唇微张，呼吸急促带动肩膀起伏，下巴微收 |
| 悔恨/愧疚 | 眉眼下垂，面部松弛，眼周微红 | 目光低垂看向地面或桌面，偶尔抬眼看对方又迅速移开 | 嘴角自然下沉，可能咬下唇，下颌微颤 |
| 愤怒/质问 | 眉头紧锁，鼻翼微扩，咬肌紧张 | 目光锐利直视对方，不眨眼或眨眼频率降低 | 嘴唇紧抿成一条线，下巴微抬，面部有克制的不平 |
| 悲伤/绝望 | 眉眼低垂，眼尾下拉，面颊松弛 | 目光涣散失焦，直视某处但"看不见"，眼皮微肿 | 嘴角下垂，面部表情沉静，丧失所有表情活动的动力 |
| 警惕/怀疑 | 眉头微皱但不紧锁，面部轻度绷紧 | 目光聚焦且稳定，眼角有观察的微动，瞳孔不变 | 头微微偏向一侧或前倾（倾听/审视姿态），嘴唇轻抿 |
| 麻木/恍惚 | 面部表情完全消失，肌肉松弛无张力 | 目光空洞，直视前方不聚焦也不移动，瞳孔无反应 | 嘴唇微微分开但不说话，面部如面具一般静止 |
| 释然/平静 | 面部线条放松，眉头舒展自然 | 目光平静温和，视线稳定但不紧张，有轻微的放远 | 嘴角有极微弱的上扬（不是笑容，是放松），呼吸平稳 |
| 犹豫/动摇 | 眉头微蹙，面部有微小的不对称活动 | 目光游移不定——看对方→看桌面→看窗外→看回对方 | 手指无意识触碰桌面/摩挲衣角/摸嘴唇，嘴巴欲言又止 |
| 决心/坚定 | 面部线条清晰，下颌微抬或微收，表情克制 | 目光笔直向前，不回避不躲闪，瞳孔稳定 | 嘴角自然闭合，下巴微收（沉稳的坚定）或微抬（决绝的坚定） |
| 愧疚/心虚 | 头部微低，眉毛中部上抬（愧疚眉），面颊微热泛红 | 目光躲闪，避免直视对方眼睛，偏向下方或侧方 | 嘴唇不安地微动/舔嘴唇，手部有无意识的小动作 |

## 二、摄影机前的光——光源逻辑与氛围

### 2.1 自然光时刻

| 时刻 | 光源行为 | 色温与色调 | 空间氛围 |
|---|---|---|---|
| 清晨 | 低角度暖白日光，通过窗户斜入室内；室外街道阳光刚越过建筑、长阴影 | 3500-4500K暖白，空气微冷 | 日常开始、尚未被侵扰的安全感、一天的起点 |
| 上午 | 日光升高，冷白清澈，阴影锐利；室内窗光面积最大 | 5000-6000K中性偏冷 | 日常运转、秩序井然、正常的办事/工作状态 |
| 正午 | 顶光或近乎顶光，阴影短而浓重；室外材质反光强 | 5500-6500K冷调 | 危险的初次信号、街头的偶遇、一切尚未察觉 |
| 午后 | 日光偏西，暖调渐强，斜光穿过窗或树形成长阴影和光斑 | 4000-5000K渐暖 | 慵懒日常、未被打破的平静、暴风雨前的宁静 |
| 金色时刻 | 极低角度暖金漫射，长阴影到极致，一切表面镀上金边 | 2800-3500K极暖 | 回忆的珍贵、家庭温情、得救后的希望——社会教育中最温暖的视觉语言 |
| 蓝调时刻 | 太阳沉入地平线后，天空深蓝紫，自然光极暗极冷，人工光源刚刚亮起 | 8000-10000K天空+暖点人工光 | 迷失/反思/独处、善恶之间的过渡、选择的路口 |
| 深夜 | 仅靠人工光源——街灯、台灯、屏幕光、车灯。大部分区域沉入暗部 | 暖光源2800-3200K / 冷光源4000-6500K | 私密的脆弱、非法活动的掩护、一个人面对的夜晚 |

### 2.2 情绪光影

| 情绪基调 | 光线配方 | 摄影约束 |
|---|---|---|
| 日常安全 | 大面积窗光漫射（4500-5500K），窗帘柔化或打开，无硬阴影 | 低反差通透感，自然真实，不做过度修饰 |
| 执法严肃 | 冷白荧光灯管顶光/侧光（5000-6500K），白墙漫反射，无暖色 | 光比适中约1:4，冷调均匀，光线"公正无私" |
| 温情回忆 | 暖调窗光/台灯侧入（2800-3500K），暗部温暖有细节 | 暖调包裹，低反差柔和，自然的高光在人物边缘 |
| 悬疑逼近 | 单光源侧打（冷白5000K或暖黄2800K），暗部浓重多阴影 | 大光比1:6以上，光源方向单一，背景沉入暗部 |
| 希望救赎 | 逆光/斜窗光（3500-4500K暖白），光线有方向感，通透 | 光比约1:2.5，暖调明亮，人物面部有受光面 |
| 深夜脆弱 | 单一暖光源（台灯2800K或窗外路灯），形成孤岛式照明 | 极简光源，暗部丰富有轮廓，光比可到1:6 |
| 对峙冲突 | 硬光侧打+深阴影，大光比，冷白光从单一方向切入 | 大光比1:8以上，光影切割面部，无柔和过渡 |
| 悔恨反思 | 单光源（台灯或顶灯）从一侧照射，半张脸沉入暗部 | 光比1:5左右，面部一明一暗，反映内心的分裂 |

## 三、摄影机前的场景——社会教育真实场景

| 场景类型 | 摄影必须捕捉的元素 |
|---|---|
| 家庭客厅 | 沙发+茶几+电视柜、窗帘窗台、茶几上的遥控器和杯具、墙面有生活痕迹（挂历/照片/贴纸残留）、地面的日常杂物——窗光是最重要的光源 |
| 派出所询问室 | 冷白荧光灯顶光、白墙+防撞软包、金属审讯桌+询问椅、角落的监控摄像头、墙面可能挂有警徽/标语、日光灯的电源开关 |
| 社区公告栏 | 铁质/木质公告栏、贴有多层宣传海报（反诈/消防/反邪教）、海报边角卷翘、透明胶带痕迹、公告栏周围可能有居民驻足 |
| 学校教室 | 课桌椅排列+使用痕迹（刻画/磨损）、黑板+粉笔灰+板书残留、窗光斜入教室、墙上通知栏、讲台+投影设备 |
| 反诈中心 | 多台显示器排列、大屏幕显示实时数据图表、工位上的电话和耳机、冷白光照明、墙上贴有工作流程图和标语 |
| 地下非法聚会点 | 昏暗室内、简易桌椅（折叠桌/塑料凳）、投影幕布+简易讲台、窗帘紧闭不透光、墙面可能有横幅或标语、地面有烟头和杂物 |
| 心理辅导室 | 暖调灯光、舒适的布艺沙发+靠垫、书架+绿植、窗帘柔和、茶几上有纸巾和水杯、墙上的暖色调装饰画 |
| 法庭 | 高高的法官席+法槌、木质法庭设施（被告席/公诉席/辩护席）、旁听席排列、庄重的国徽、严肃的冷光或混合光 |
| 城市深夜街道 | 湿路面反射街灯（如有雨）、空荡的街道+孤零零的路灯、打烊的店铺+卷帘门、远光灯扫过的车灯轨迹 |
| 医院/急救室走廊 | 日光灯冷白长条照明、白墙+扶手、等候区的塑料椅、墙上贴有医院科室指示牌、消毒水的清洁感与冷感 |

## 四、摄影机语法——镜头选择与运动

### 4.1 焦段选择

| 焦段 | 叙事性格 | 适用场景 |
|---|---|---|
| 24-28mm广角 | 环境包围人物、纪实感、揭示全貌 | 审讯室全景、家庭全貌、社区环境、警示空间展示 |
| 35mm小广角 | 主观视点、身临其境的纪实感 | 纪实跟随、家庭对话、街头走访、询问过程 |
| 50mm标准 | 人眼等价、客观中立的注视 | 人物肖像、受害者/违法者表情记录、对话中的情感凝视 |
| 85mm中长焦 | 压缩空间、拉近距离、捕捉不易察觉的微表情 | 特写捕捉、远距离观察、情绪爆发的瞬间 |
| 135mm+长焦 | 将人物从环境中剥离、窥视感 | 暗中观察/跟踪视角、被监视感、社会边缘个体的远眺 |

### 4.2 景别选择

| 景别 | 取景范围 | 叙事功能 |
|---|---|---|
| 大特写 | 眼睛/手指/手机屏幕/桌面关键物 | 情绪爆发点、关键信息揭示（诈骗短信/关键证据）、感官聚焦 |
| 近景 | 头顶至胸部以上 | 人物情感、对话反应、面部微表情——社会教育的情感核心 |
| 中近景 | 头顶至腰部 | 人物互动、手势动作、对话中的身体语言（如交出手机的动作） |
| 中景 | 头顶至膝盖上下 | 人物与空间的初级关系（嫌疑人和审讯桌、受害者和家） |
| 全景 | 人物全身+部分环境 | 人物在空间中的位置、执法与被执法者的权力关系 |
| 大全景/空镜 | 空间全貌，人物极小或无人物 | 空间叙事、情绪缓冲、隐喻留白 |

### 4.3 摄影机运动

| 运动方式 | 视觉心理 | 使用场景 |
|---|---|---|
| 固定 | 冷静观察，客观记录——社会教育的第一运动方式 | 对话、询问、审讯、空镜留白 |
| 手持微晃 | 在场感，呼吸感，主观介入 | 情绪波动、被骗后的慌乱/奔跑、冲突时刻 |
| 稳定器流动 | 平滑移动，揭示全貌 | 空间展示（从外到内）、人物入场、过渡镜头 |
| 缓推 | 注意力逐渐聚焦、情绪趋紧 | 逼近真相、关键信息的揭示、情绪爆发的前奏 |
| 缓拉 | 拉开距离、揭示全貌、情绪抽离 | 揭示环境全貌/后果、结尾留白、教育信息的收束 |
| 跟拍 | 与人物同行、纪实跟随 | 街头走访、被带走/被追踪、日常行走 |
| 摇镜 | 视线转移、空间关系交代 | 从对话一方移至另一方、从人物移至环境/证据 |

## 五、固定风格锚定词

**摄影锚定（必选）：**

真人实拍摄影，社会教育纪实影像，当代中国社会空间，真人实拍质感，自然光与人造光调度，真实色彩科学，纪实摄影风格

**人物质感（含人物镜头时必选）：**

真人写实摄影质感，真实皮肤纹理毛孔可见，不磨皮不硅胶脸，发丝自然质感有碎发，素颜或极淡妆状态，日常便服面料纹理可见，自然微表情与肢体语言，大众日常体态

**场景质感（含场景镜头时必选）：**

真人实拍社会场景摄影，真实空间材质和岁月痕迹（墙面斑驳/地面磨损/家具使用痕迹），光源方向明确可追溯，生活痕迹自然可见，非样板间非效果图

**参考图一致性锚定（参考图模式必选）：**

保持人物造型、面容、发型、服饰与参考图完全一致，保持场景风格与参考图一致，保持光影色彩基调统一，所有镜头呈现为同一人物的连续摄影记录

**风格收尾（固定）：**

真人写实摄影，当代中国社会教育电影质感，纪实级摄影画面，非3D非渲染非CG

**画质锁定词：**

模式A（中文）——默认：
真人摄影高清画质，纪实级画面质感，画面无字幕、无水印、无标题叠字、无UI元素

模式A（中文）——画内文字场景（画面中含招牌/路牌/屏幕/书籍等道具文字时）：
真人摄影高清画质，纪实级画面质感，画面无字幕、无水印、无标题叠字、无UI元素，招牌/路牌/屏幕/书本等场景道具上的文字清晰可辨

模式B（英文）——默认：
live-action documentary photography, real human skin texture, contemporary Chinese social environment, natural light photography, high detail, no subtitles, no captions, no watermark, no title overlay, no UI elements

模式B（英文）——画内文字场景：
live-action documentary photography, real human skin texture, contemporary Chinese social environment, natural light photography, high detail, no subtitles, no captions, no watermark, no title overlay, no UI elements, legible text on in-scene props such as signboards, screens and documents

**负向词模板（模式B 必须包含，置于提示词末尾）：**

> ⚠️ Seedream（模式A）不支持负向提示词，负向词仅适用于模式B。模式A 通过正向词中的质感锚定和画质锁定来保证画面质量。

模式B（英文）：
no 3D render, no CGI, no Unreal Engine, no Blender, no PBR material, no game engine, no 2D cartoon, no anime, no illustration, no hand drawn, no painting, no plastic skin, no wax face, no airbrushed, no poreless skin, no silicone face, no ancient Chinese style, no traditional costume, no cyberpunk, no sci-fi, no western fantasy, no medieval, no non-Chinese urban, no horror movie lighting, no plastic mannequin, no symmetrical model pose, no branded new clothes, no showroom, no wrong anatomy, no deformed face, no broken limbs, no distorted body, no subtitles, no captions, no watermark, no title overlay, no UI text

## 六、美学禁止项

以下词汇/风格不得出现于输出提示词中：

- ❌ 3D渲染 / 3D建模 / CG / UE引擎 / Blender / PBR材质 / 次世代建模 / 体积光 / 环境光遮蔽 / 光线追踪
- ❌ 2D手绘 / 插画 / 动画 / 二次元 / 漫画 / Q版 / 低多边形
- ❌ 写实摄影（英文禁止 photorealistic / realistic photography——社会教育是"真人实拍摄影"live-action documentary photography，不是"拟真渲染"）
- ❌ 过度磨皮 / 塑料假面 / 零毛孔 / 硅胶脸 / 蜡像感 / AI感平滑
- ❌ 古风 / 古装 / 汉服 / 仙侠 / 武侠 / 民国 / 赛博朋克 / 科幻 / 西方奇幻 / 中世纪
- ❌ 恐怖片式光影 / 刻意阴间色调 / 鬼片氛围 / 过度血腥/暴力
- ❌ 美化/浪漫化/神秘化邪教、诈骗、违法犯罪
- ❌ 人体结构错误 / 肢体畸形 / 面部崩坏 / 比例失调
- ❌ 高饱和荧光色 / 廉价滤镜感
- ❌ 非中国当代社会场景 / 无中国特征的架空空间
