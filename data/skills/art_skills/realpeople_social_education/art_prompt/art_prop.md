# 真人实拍社会教育道具图像生成 · 约束手册

---

## 一、道具设计原则

> 社会教育的道具不是"被建模的物体"，而是"在中国社会事件中被使用过的物品"——它是某个人在某个关键社会情境中接触过的物证。摄影机拍下的是它的此刻。

1. **物品即社会证据** — 每一件道具都是一个社会教育叙事的物证。手机上的诈骗短信、邪教的宣传册、反诈中心的传单——道具自己会说话
2. **使用痕迹优先于完美品相** — 社会教育的道具必须有"被使用过"的证据。全新的、刚拆封的、样板级别的道具违背了社会教育叙事的真实感要求
3. **中国当代社会真实物品** — 道具必须是中国社会生活中真实存在的物品：国产手机品牌+中文界面+中国APP界面、中文包装、中国式宣传物料——拒绝无国别的架空物品
4. **材质即真实** — 不是PBR参数渲染出的材质，而是摄影机拍下的真实材质行为：纸张的纹理、塑料的注塑纹理、金属的氧化痕迹
5. **单视图静物摄影** — 道具以单张静物产品摄影方式独立呈现，非多角度建模展示，非2×2四宫格

## 三、单视图静物摄影规范

> 社会教育道具仅需单张静物产品摄影，非多角度建模展示。以下为单视图摄影的完整规范。

### 画面构成

| 项目 | 摄影要求 |
|---|---|
| 道具呈现 | 道具居中或偏三分线位置独立陈列，完整入画，无裁切 |
| 背景 | 素灰无缝背景纸 #B0B0B0（哑光），或纯色桌面（浅木纹/米白桌面/深色办公桌面）。无多余物体、无装饰、无文字 |
| 光线 | 自然窗光或单灯柔光。窗光45°侧入（明亮柔和带自然阴影）、或柔光箱顶+侧补。光影柔和、方向明确 |
| 景深 | 浅景深（f/2.8-f/5.6），道具主体清晰锐利，前后微虚化 |
| 投影 | 道具底部与承载面之间的自然接触阴影——真实的而非悬浮。阴影柔软、方向与主光源一致 |
| 画面比例 | 建议 1:1 正方形或 4:3 |

### 道具状态准则

| 维度 | 要求 |
|---|---|
| 使用痕迹 | 必须有"被使用过"的视觉证据——非全新未拆封、非样板品相 |
| 中国辨识度 | 中文标识/中国品牌/中国式包装/中国式宣传物料 |
| 材质真实感 | 非渲染质感——是摄影机拍下的真实材料行为 |
| 社会教育功能 | 道具的叙事指向明确——服务于警示教育的视觉证据 |

## 四、材质与表面——摄影机下的真实物体

| 材质 | 在摄影机下的真实表现 | 提示词 |
|---|---|---|
| 金属（不锈钢/铝合金） | 拉丝肌理可见、边缘高光锐利但不死白、表面有指纹和微划痕 | 金属拉丝肌理、环境反射自然、表面微划痕与指纹 |
| 玻璃（手机屏/相框） | 通透但有自身微弱颜色、表面有指印或污渍、边缘有光折射 | 玻璃自然通透、表面真实指印、非CG完美透明 |
| 塑料（外壳/包装） | 注塑纹理/合模线可见、表面细微划痕、哑光或半哑光 | 注塑纹理清晰、合模线自然可见、表面使用痕迹 |
| 纸质（宣传单/文件） | 纤维粗糙度可见、折痕自然、印刷文字微洇、边缘翻阅磨损 | 纸质纤维纹理、自然折痕、印刷文字真实质感 |
| 皮革（钱包/证照套） | 天然纹理/毛孔可见、弯折处使用痕迹、边缘自然磨损 | 皮革天然纹理、弯折使用痕迹、边缘自然磨损 |

## 六、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须是「单张静物摄影」——单视图、非多角度、非2×2四宫格、非建模展示 |
| R2 | 必须指定「素灰背景纸 #B0B0B0」或「纯色桌面」等真实表面，禁止复杂场景 |
| R3 | 必须指定光源逻辑——窗光或柔光箱，光影方向明确、投影自然 |
| R4 | 道具必须有「使用痕迹」——非全新、非样板品相、非未拆封 |
| R5 | 道具必须具备中国当代社会辨识度——中文标识/中国品牌/中国式宣传物料 |
| R6 | 必须声明「实物产品摄影 + 非3D渲染非CG」核心锚定 |
| R7 | 材质必须描述为摄影机下的真实表现，而非渲染参数 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「3D渲染 / 3D建模 / CG / UE引擎 / Blender / PBR材质」等一切CG术语 |
| X2 | 严禁「2D手绘 / 插画 / 动画 / 二次元」等非摄影媒介 |
| X3 | 严禁「多角度 / 四视图 / 2×2网格 / 正交视图 / turnaround / blueprint」——仅单张 |
| X4 | 严禁「全新未拆封 / 样板品相 / 无使用痕迹 / 完美品相」的道具 |
| X5 | 严禁「古风/古代/仙侠/武侠/民国/赛博朋克/科幻/西方奇幻/架空」等非当代社会物品 |
| X6 | 严禁「无中国辨识度的架空物品 / 全英文包装 / 非中国品牌」 |
| X7 | 严禁「出现人物/手部/手指/肢体」 |
| X8 | 严禁「悬浮/抠图白底/无投影/无承载面」——物品必须放在真实表面上 |
| X9 | 严禁通过道具美化/浪漫化邪教、诈骗、违法犯罪行为 |
| X10 | 严禁「水印 / 文字 / LOGO / 签名 / 边框 / AI生成痕迹」 |
