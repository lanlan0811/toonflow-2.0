# 真人实拍社会教育分镜视频生成 · 约束手册

---

## 一、视频生成提示词规范

> 社会教育分镜视频的提示词生成，遵循"将分镜表转化为AI可执行的具象视频描述"的核心原则。所有描述必须是物理可执行的，而非抽象的情感概括。

### 1.1 视频提示词与图像提示词的区别

| 维度 | 图像提示词 | 视频提示词 |
|---|---|---|
| 时间维度 | 单帧静态描述 | 须描述动作的轨迹/速度/时长/先后顺序 |
| 光源描述 | 光源方向和色温 | 光源在时间中的连续性——光是否变化、何时变化 |
| 人物状态 | 静态表情和体态 | 表情的微变化过程、体态的转换轨迹 |
| 环境动态 | 静态环境 | 环境的动态——窗帘飘动/灯光闪烁/人影走动 |
| 转场 | 不涉及 | 前后镜头的视觉衔接方式 |

### 1.2 视频提示词结构

一个完整的社会教育分镜视频提示词应包含以下要素：

1. **画质底座** — 锁定真人实拍摄影品质
2. **角色锚定** — 指定角色参考图及一致性要求
3. **场景锚定** — 指定场景参考图及环境要求
4. **人物动态** — 动作轨迹+时长+身体部位协调
5. **微表情描述** — 含眼神/面容/情绪的细微变化
6. **光源描述** — 主光方向+色温K+光比+光源变化
7. **环境动态** — 背景中的动态元素
8. **服装/道具动态** — 与动作相关的服饰表现
9. **负面词** — 规避非真人质感

## 二、时序控制规范

### 2.1 社会教育短片的典型时长规划

| 段落 | 时长 | 内容 | 节奏 |
|---|---|---|---|
| 开场/日常建立 | 5-10秒 | 日常场景、角色正常状态、安全的"之前" | 平缓，固定机位为主 |
| 危机介入 | 5-8秒 | 收到诈骗信息/被邪教接触/陷阱的开始 | 稍加速，可选手持微晃 |
| 过程/被控制 | 10-20秒 | 被骗/被控制的过程、受害者的心理变化 | 中速，固定+手持交替 |
| 警示/揭穿 | 5-10秒 | 真相揭示/执法介入/反诈中心介入 | 加速，多个短镜切换 |
| 后果/反思 | 8-15秒 | 受害者的悔恨、家人的悲痛、违法者的后果 | 缓慢，长镜头为主 |
| 希望/结尾 | 5-10秒 | 获得帮助/重新开始/警示字幕前的最后画面 | 舒缓，暖调收束 |

### 2.2 单镜头时长规范

- **固定镜头**：3-6秒（社会教育最常用的镜头时长，让观众有时间阅读人物情绪）
- **运动镜头**：4-8秒（缓推/缓拉需要有足够的时长让观众感知运动）
- **手持镜头**：3-5秒（手持过长会引起不适，控制在情绪节奏需要的范围内）
- **特写镜头**：2-4秒（特写是高信息密度镜头，时间过长会压迫观众）
- **空镜**：3-6秒（大于3秒才能形成"空镜的情绪缓冲"效果）

## 三、动作连续性规范

### 3.1 社会教育短片的动作连续性原则

| 原则 | 说明 |
|---|---|
| 动作轨迹完整 | 每个动作描述"从哪里开始→经过什么路径→在哪里结束" |
| 身体部位协调 | 一个动作涉及哪些身体部位，它们如何配合 |
| 速度与节奏 | 动作是快是慢——紧张的场景动作偏快，反思的场景动作偏慢 |
| 对手部动作特别关注 | 社会教育中手部动作承载大量叙事信息——交出手机/签字/掩面/握拳 |

### 3.2 常见社会教育动作的连续性描述

| 动作 | 分镜视频连续性描述 |
|---|---|
| 看手机（发现诈骗信息） | 右手从桌面拿起手机→拇指点击屏幕唤醒→目光从屏幕顶端逐行下移→约2秒后眉头皱起→左手不自觉地握拳→嘴巴微张→缓缓放下手机到桌面（约1秒）→目光仍停留在手机屏幕上约2秒 |
| 在询问室等待 | 坐在询问椅上→双手交握放在桌面→目光低垂→约3秒后缓缓抬头看向门口→约2秒后低头回到原位→右手食指在桌面无意识地画圈约2秒 |
| 签字（悔过书） | 左手按住纸张上沿→右手拿起签字笔→笔尖接触纸张→缓慢写下第一个字约2秒→停顿1秒→继续书写→笔尖离开纸张约0.5秒后放下笔→右手回到桌面 |

## 五、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 视频提示词必须包含具体的动作轨迹、时长和身体部位协调——禁止"然后""随之"等模糊连接词 |
| R2 | 光源描述必须包含色温K值和光比——光线变化必须标注触发事件 |
| R3 | 必须使用@reference语法锚定角色、场景、核心道具的视觉一致性 |
| R4 | 画质底座必须声明「真人实拍摄影 + 非3D非渲染非CG」 |
| R5 | 叙事型镜头时长控制在3-6秒，专门的情绪留白镜头可延长至8秒 |
| R6 | 每3-4个镜头至少安排一个有环境动态的镜头 |
| R7 | 社会教育内容必须包含"警示或教育目的"的视觉提示——不纯为展示而展示 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「3D渲染 / CG / UE引擎 / Blender / PBR材质」等一切CG术语 |
| X2 | 严禁「恐怖片式光影/音效/运镜——社会教育不等于恐怖片」 |
| X3 | 严禁「美化/浪漫化邪教、诈骗、违法犯罪行为」 |
| X4 | 严禁「无叙事目的的快速甩镜/花式转场/特效转场」 |
| X5 | 严禁「动作无轨迹无时长——'然后站起来'等模糊描述」 |
| X6 | 严禁「光源无来源——画面亮但说不清光从哪来」 |
| X7 | 严禁「古风 / 非当代 / 赛博朋克 / 科幻」等非当代社会场景 |
| X8 | 严禁「水印 / 字幕 / LOGO / UI元素 / 生成工具痕迹」 |
