# 真人实拍社会教育场景图像生成 · 约束手册

---

## 一、场景美学原则

> 社会教育场景不是"被建模的空间"，而是"被摄影机选中的真实中国社会场所"。没有渲染引擎、没有贴图、没有体积光参数——只有实际存在的光、真实的材料、以及中国社会的日常。

1. **空间即社会叙事** — 一个空无一人的询问室不是"空的"，是"刚审讯完"。一把歪放的询问椅、桌上半瓶没喝完的矿泉水、角落里监控摄像头的指示灯闪烁——空间自己会讲社会教育的故事
2. **真实的光，不是渲染的光** — 场景的光只有一个来源：实际的日光（通过窗户、门洞）、实际的灯光（荧光灯管、台灯、街灯、日光灯）。光的色温和方向必须在物理上说得通
3. **被使用过的社会空间** — 真实的场景不整洁。派出所询问室的白墙有磨损、家庭客厅的沙发有坐痕、社区公告栏的海报有卷角和褪色——这些使用痕迹是"真实社会空间"的证据
4. **中国当代社会的真实场所** — 场景必须有中国社会辨识度——派出所的标识和警徽、社区的宣传栏和条幅、家庭的陈设风格、学校的教室布置。这不是"generic space"，这是"当代中国社会的一个具体角落"
5. **单视图全景全貌** — 一张照片装下整个场地的完整信息。广角或标准焦段，全景深，正面展示空间的全貌与纵深——不是四视图分割，不是多角度拼贴

## 二、季节与时刻——自然光的导演

> 社会教育场景的光，首先由季节和时刻决定。以下是真实光源在四个季节、五个时段的摄影行为。

| 季节 | 时段 | 光的行为 | 场景氛围词 |
|---|---|---|---|
| 春 | 清晨 6:00-8:00 | 低角度暖白日光、空气中有薄雾的漫射、建筑表面柔和 | 春日晨光乍泄、街区静谧未醒、日常尚未被侵扰 |
| 春 | 正午 11:00-13:00 | 日光近乎垂直、阴影短、建筑表面材质清晰 | 明亮暴露、一切无所遁形——适合揭示真相的场景 |
| 春 | 午后 14:00-16:00 | 日光偏西、暖调渐强、长阴影开始形成 | 日常的午后、暴风雨前的平静 |
| 春 | 金色时刻 17:00-18:30 | 暖金低角度光、长阴影、一切表面镀金边 | 珍贵的温暖——家庭记忆、得救后的希望 |
| 春 | 夜晚 19:00-次日 | 街灯/室内灯为主、冷暖光并存 | 春夜微凉、夜色中的隐秘与反思 |
| 夏 | 清晨 5:00-7:00 | 极早的暖光、空气清澈、长阴影 | 夏季晨光——被骗的一天开始前的宁静 |
| 夏 | 正午 11:00-14:00 | 顶光强、阴影浓重短促、路面热浪隐约 | 盛夏的烈日——街头相遇/被骗发生的时刻 |
| 夏 | 金色时刻 18:00-19:30 | 极暖调低角度光、长阴影、城市镀暖橙金 | 夏日金色黄昏——回忆中最温暖的光 |
| 夏 | 夜晚 20:00-次日 | 街灯/车灯为主、暖光与冷光并存 | 夏夜的迷离与不安 |
| 秋 | 清晨 6:30-8:00 | 清冷暖白日光、空气干燥通透 | 秋日清冽——清醒、冷静、适合做决定的时刻 |
| 秋 | 金色时刻 16:30-18:00 | 极暖金橙光、一年中最浓郁的金色 | 熔金秋色——警示最后的温暖、新生的象征 |
| 秋 | 夜晚 18:30-次日 | 入夜早、街灯+车灯为主 | 秋夜微寒——独处反思的季节 |
| 冬 | 清晨 7:00-8:30 | 低角度冷白日光、空气冷硬 | 冬日晨光——冷冽中的一线希望 |
| 冬 | 正午 11:00-13:00 | 日光偏南低角度、全天最暖的时刻 | 冬日暖阳——难得的温暖、珍贵的光 |
| 冬 | 夜晚 17:00-次日 | 入夜极早、街灯/橱窗光/车灯为主 | 冬夜漫长——寒夜中的孤独与等待 |

## 三、社会教育场景类型

### 3.1 家庭/居住空间

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 普通家庭客厅 | 布艺沙发+茶几、电视柜+电视、窗帘+窗台杂物、地板/地砖、墙面有生活痕迹（挂历/照片/贴纸残留）、茶几上的遥控器和水杯 | 日常温馨、普通中国家庭的真实状态、未被侵扰的日常 |
| 家庭卧室 | 床+床品、床头柜+台灯/充电器、衣柜+衣物、窗帘、个人杂物的自然散落 | 私密真实、最脆弱也最安全的个人空间 |
| 家庭厨房 | 灶台+油烟机、碗柜+餐具、餐桌+餐椅、油盐酱醋瓶罐、窗户+窗台杂物 | 日常烟火气、家庭的温暖核心、日常对话的发生地 |
| 老小区楼道 | 水泥/地砖楼梯+扶手、各家门口的不同鞋垫/杂物、墙面斑驳+小广告、声控灯、楼道窗户 | 市井生活肌理、被岁月磨过的公共空间 |

### 3.2 执法/司法空间

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 派出所询问室 | 金属询问椅+审讯桌、冷白荧光灯顶光、白墙（可能有防撞软包）、角落监控摄像头、墙面标语 | 严肃肃穆、冷白光线无所隐藏、正义与真相的场所 |
| 派出所接待大厅 | 柜台+办事窗口、等候区塑料椅、公告栏+宣传海报、警徽+标语、冷光照明 | 开放但严肃、公共安全的第一道门 |
| 反诈中心 | 多台显示器排列、大屏幕数据可视化、工位+电话/耳机、冷白光+屏幕光 | 科技反制、现代执法、与时间赛跑的战场 |
| 法庭 | 法官席高位+法槌+国徽、被告席+旁听席、木质陈设、庄重照明 | 法律终裁、庄严肃穆、正义的裁决地 |

### 3.3 社区/公共空间

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 社区公告栏/宣传栏 | 铁质/木质公告栏、多层宣传海报（反诈/消防/反邪教）、海报边角卷翘、周围可能有居民驻足 | 社会教育的基层阵地、日常中的警示提醒 |
| 居民小区 | 居民楼+阳台、健身器材+长椅、行道树+绿化、路灯+单元门、停放的电动车/自行车 | 最真实的中国社区日常、邻里生活的基础单元 |
| 社区活动中心 | 会议桌+折叠椅、投影幕布+讲台、墙上标语+宣传画、日光灯照明 | 社区教育的发生地、讲座/宣传活动的场所 |
| 学校/教室 | 课桌椅排列+使用痕迹、黑板+粉笔灰残留、窗光斜入、墙上通知栏+黑板报 | 青少年教育的阵地、青春与知识的空间 |

### 3.4 城市公共空间（社会教育叙事）

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 城市街道/十字路口 | 斑马线+红绿灯、行道树+路灯、沿街店铺+广告招牌、行人+车流 | 日常中的陷阱可能发生的场所 |
| 地铁站/车厢 | 闸机+安检、站台+屏蔽门、广告灯箱、车厢座椅+扶手 | 都市流动空间——诈骗/邪教接触的高频场所 |
| 银行/ATM自助区 | ATM机+操作面板、监控摄像头+安全提示、冷光照明、金属/玻璃结构 | 转账/被骗发生的关键场景 |
| 公园/广场 | 长椅+步道、树木+草坪、凉亭+健身区、晨练/休闲的居民 | 老年人的社交空间——针对老年诈骗/邪教渗透的场所 |

### 3.5 特殊教育场景

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 非法聚会点 | 昏暗室内、简易桌椅（折叠桌/塑料凳）、投影幕布+讲台、窗帘紧闭、横幅/标语 | 压抑封闭——邪教/非法传销的典型空间 |
| 心理辅导室/救助站 | 暖调灯光、布艺沙发+靠垫、书架+绿植、柔和窗帘、茶几+纸巾盒 | 温暖治愈——与社会教育机构形成对比的救助空间 |
| 拘留所/看守所 | 铁门+铁窗+灰色墙壁、狭小空间、日光灯冷光、简单的床铺+桌椅 | 警示教育的终点、失去自由的视觉证明 |

## 四、单视图全景摄影规范

> 社会教育场景仅需**单张**全景摄影——一张照片装下整个空间的完整信息。不是2×2四视图，不是多角度拼接。

### 画面构成

| 项目 | 摄影要求 |
|---|---|
| 空间呈现 | 单张照片完整呈现场景的空间全貌。使用24-28mm广角或35mm，全场地的纵深、宽度、高度在一张画面中全部可见 |
| 构图 | 全景深大景别——前景（门框/桌椅/护栏/公告栏）形成天然画框→中景是场地主体→后景向空间纵深/窗外延伸 |
| 景深 | 全景深 f/8-f/11，从前到后全部清晰。社会教育场景的每一个角落都应该能被看见 |
| 人物 | **严禁出现任何人物、人影、人体轮廓。** 场景是空的——但这种"空"有叙事：刚结束询问/即将开庭/人去楼空/日常的静止瞬间 |
| 比例 | 建议 16:9 或 3:2 宽幅——匹配全景空间展示的横向视觉习惯 |
| 色彩 | 由实际光源的色温驱动，不套滤镜。色彩科学参考纪实摄影色彩标准 |

### 空的场景如何叙事

| "空"的类型 | 画面线索 | 叙事暗示 |
|---|---|---|
| 刚结束询问 | 询问椅没有推回原位、桌上有一杯没喝完的水、一张空白的悔过书和笔、监控摄像头指示灯亮着 | 刚才有人在这里接受询问，现在短暂离开 |
| 即将开庭 | 法庭的旁听席空着、法官席的文件已摆好、灯光全亮 | 一场审判即将开始，现在是开场前的安静 |
| 人去楼空（被骗后） | 客厅沙发上有一件随手搭着的外套、茶几上的半杯茶已凉、灯亮着但没有人 | 被诈骗/被带走后的空荡家庭 |
| 日常的静止 | 社区公告栏的海报被风吹起一角、宣传单在桌上摊开、光从窗户斜入打在地板上 | 社会教育信息等待被阅读的状态 |
| 深夜的场所 | 空荡的派出所走廊只有日光灯亮着、反诈中心的大屏幕还在闪烁数据 | 城市的安全守护者还醒着 |

## 六、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须是**单张**全景摄影——单视图、非2×2四视图、非多角度、非turnaround |
| R2 | 必须使用广角镜头（24-28mm）或标准焦段（35mm），全景深 f/8-f/11，完整空间纵深 |
| R3 | 光源必须可追溯——自然光的方向与时刻匹配、人工光源的位置和色温合理 |
| R4 | 场景必须有前/中/后景空间层次，前景画框→中景主体→后景延伸 |
| R5 | 必须有中国当代社会辨识度——派出所标识/社区宣传栏/家庭陈设/中文标识等 |
| R6 | 必须有使用痕迹——墙面/地面/家具的自然磨损、真实社会空间的使用证据 |
| R7 | **严禁出现任何人物、人影、人体轮廓、剪影** |
| R8 | 必须声明"真人实拍摄影 + 非3D渲染非CG"核心锚定 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「3D渲染 / 3D建模 / CG / UE引擎 / Blender / PBR材质 / 体积光 / AO」等一切CG术语 |
| X2 | 严禁「2D手绘 / 插画 / 动画 / 二次元」等非摄影媒介 |
| X3 | 严禁「四视图 / 2×2网格 / 多角度 / turnaround / orthographic view / blueprint」——仅单张 |
| X4 | 严禁「古风/古代/仙侠/武侠/民国/赛博朋克/科幻/西方奇幻/异世界」等非当代社会场景 |
| X5 | 严禁「无中国辨识度的架空场景 / 全英文标识」 |
| X6 | 严禁「出现任何人物/人影/人体轮廓/剪影/肢体」 |
| X7 | 严禁「样板间/售楼处效果图/全新无痕/完美品相/无生活痕迹」的场景 |
| X8 | 严禁「光源来源不明/全局均匀照明/无方向的光」 |
| X9 | 严禁「过曝死白/暗部死黑/无景深/画面扁平」 |
| X10 | 严禁「水印 / 文字 / LOGO / 签名 / 边框 / AI生成痕迹」 |
