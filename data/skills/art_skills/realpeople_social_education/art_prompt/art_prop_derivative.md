# 真人实拍社会教育道具衍生 · 约束手册

---

## 一、道具衍生原则

> 社会教育道具的衍生不是"换一款产品"，而是"同一件物品在不同叙事阶段的状态变化"。核心原则：**同一件物证，不同的命运阶段**。

1. **物品一致性优先** — 所有衍生版本必须是同一件道具的不同状态——同一部手机被骗前vs被骗后、同一张宣传单在发放时vs被丢弃后
2. **状态变化反映叙事进展** — 道具的物理状态变化（变旧/破损/被使用）映射叙事的时间流逝和事件发展
3. **变化有叙事逻辑** — 道具的变化必须可以被叙事解释——屏幕碎裂是因为被摔、纸张褶皱是因为被揉过、污渍是因为被雨水打湿

---

## 二、使用状态变体

### 全新/初始状态

道具的初始状态：使用前/购买时/发放时的状态。已拆封但尚未被密集使用，有轻微全新痕迹但仍是"即将投入使用"的状态。

### 正常使用状态

道具在日常使用中的状态：屏幕亮着显示内容、证件有正常翻折痕迹、宣传单已被阅读有折痕。这是社会教育叙事中最常见的道具状态。

### 受损/使用后状态

| 道具类型 | 受损状态描述 |
|---|---|
| 手机 | 屏幕碎裂（被摔/被砸）、外壳磕碰明显、屏幕不亮或碎裂但部分显示 |
| 银行卡/身份证 | 被剪断/折断（销户/挂失的视觉证据）、表面严重划痕 |
| 宣传单/文件 | 被揉皱/被撕破/被雨打湿字迹洇开、边角严重卷翘 |
| 相框/照片 | 相框玻璃碎裂、照片被撕开又粘合、边角烧焦痕迹 |
| 书包/衣物 | 被撕破/被扯坏、肩带断裂、有残留污渍 |

## 三、场景变体

### 在不同社会教育场景中的状态

| 场景 | 道具状态 |
|---|---|
| 派出所/执法机构 | 被当作证物——证物袋包装+标签、文件有印章+编号、物品作为案件证据被摆放在桌面上 |
| 家庭/日常 | 日常使用状态——随意放在茶几/桌面/床上、未特别摆放的日常状态 |
| 社区宣传 | 宣传物料被张贴在公告栏（有透明胶带/图钉固定）、宣传单被拿在手中或被叠放成堆 |

## 五、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有衍生变体必须是同一件道具的不同状态——核心物品可辨认 |
| R2 | 状态变化必须有叙事逻辑——受损/使用的程度可用叙事解释 |
| R3 | 必须声明「单张静物摄影 + 非3D非渲染非CG」 |
| R4 | 必须声明「中国当代社会真实物品」 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「不同变体变成完全不同的道具——核心物品不一致」 |
| X2 | 严禁「3D渲染 / CG / 后期合成的损坏效果——损坏应该是真实的」 |
| X3 | 严禁「出现人物/手部/手指/肢体」 |
| X4 | 严禁「通过道具受损美化/浪漫化暴力或违法行为」 |
| X5 | 严禁「古风 / 非当代 / 架空」等非当代中国物品 |
