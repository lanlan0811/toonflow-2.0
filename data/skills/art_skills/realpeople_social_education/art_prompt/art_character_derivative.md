# 真人实拍社会教育角色衍生 · 约束手册

---

## 一、角色衍生原则

> 社会教育角色的衍生不是"换皮肤"，而是"一个人在不同社会情境下的状态变化"。衍生的核心是：**同一张面孔，不同的命运阶段**。

1. **身份一致性优先** — 所有衍生版本必须是同一个人的不同状态。面容核心特征（颌骨轮廓、鼻梁转折、耳廓形状、发际线走向、眼神气质）必须保持稳定可辨认
2. **变化驱动叙事** — 每个衍生变体服务于社会教育叙事：受害前→受害中→受害后→得救后，每一阶段的光影、状态、着装都有其叙事功能
3. **变化有因果** — 人物状态的变化必须可以被叙事解释——憔悴是因为被骗后失眠、凌乱是因为经历了冲突、服装变化是因为季节/场景变化
4. **不美化堕落** — 反面角色（违法者/诈骗犯/邪教成员）的状态变化不应呈现"有魅力"的堕落，而是呈现"普通人的误入歧途"——通过状态变化传递警示而非神秘化

## 二、情绪状态变体

### 正常/日常状态

| 维度 | 描述 | 提示词 |
|---|---|---|
| 面容 | 日常状态，面部肌肉放松，表情自然中性 | 面部表情自然放松、日常状态、未被事件影响的普通人 |
| 眼神 | 自然注视，目光稳定，不闪躲不涣散 | 自然稳定的注视、日常状态的眼神 |
| 体态 | 日常站姿/坐姿——重心正常、体态自然放松 | 日常自然体态、放松的站姿/坐姿 |
| 皮肤 | 正常肤色和光泽，无明显憔悴迹象 | 日常皮肤状态、正常肤色和光泽 |
| 发型 | 日常整洁发型，自然打理 | 日常整洁发型、自然的日常状态 |
| 服装 | 日常基础服装 | 素色基础款日常服装 |

### 憔悴/受创状态

| 维度 | 描述 | 提示词 |
|---|---|---|
| 面容 | 面部肌肉松弛，眼窝微陷，面部出现疲惫纹路（法令纹加深、眉间有竖纹）、面色暗沉 | 面部疲惫松弛、眼窝微陷、法令纹加深、面色暗沉——被事件摧残的痕迹 |
| 眼神 | 目光涣散或低垂，眼皮微沉，眼周微红（哭过或失眠），黑眼圈可见 | 目光涣散低垂、眼周微红、可见黑眼圈、失眠与被摧残的眼神 |
| 体态 | 低头含胸、肩内收、身体微缩、动作缓慢 | 低头含胸、身体微缩、动作迟缓疲惫 |
| 皮肤 | 暗沉无光泽，眼周暗沉明显，可能出现油光与暗沉并存的不健康肤感 | 皮肤暗沉无光、眼周明显暗沉、不健康的皮肤质感 |
| 发型 | 头发微乱——可能被汗浸湿贴头皮、碎发散乱、发尾毛躁 | 头发微乱、可能被汗浸湿、碎发散乱——没有精力打理的状态 |
| 服装 | 同一套服装但呈现皱褶增多——可能穿着同一件衣服持续多日 | 同一件衣服出现更多的褶皱——没换衣服/没力气打理 |

### 震惊/恐惧状态

| 维度 | 描述 | 提示词 |
|---|---|---|
| 面容 | 面部微绷，眉毛上抬，额头出现横纹，嘴巴微张或紧抿 | 面部紧绷、眉毛上抬、微张或紧抿的嘴唇——震惊中的面容 |
| 眼神 | 瞳孔微扩，目光聚焦在某个方向但不移动，快速眨眼 | 瞳孔微扩、目光定在某处不移动、快速眨眼——恐惧的注视 |
| 体态 | 身体后倾或僵住不动，双手可能握拳或捂住嘴巴，肩膀微耸 | 身体僵住/后倾、双手握拳或捂嘴、微耸的肩膀——受惊的体态 |
| 皮肤 | 面部可能泛白（血色褪去），T区细微汗珠可见 | 面部微泛白、T区细微汗珠——身体应激反应 |
| 服装 | 服装同日常状态，可能因手部动作导致衣摆被攥出皱褶 | 服装同日常状态、可能被手攥出皱褶 |

### 悔恨/愧疚状态

| 维度 | 描述 | 提示词 |
|---|---|---|
| 面容 | 眉眼下垂，面部松弛，嘴角向下，眉头微蹙——"后悔眉"的典型表现 | 眉眼下垂、嘴角向下、眉头微蹙——悔恨的面容、愧疚的表情 |
| 眼神 | 目光低垂或游移——不敢直视他人，偶尔抬眼又迅速移开 | 目光低垂游移、不敢直视、回避的眼神——愧疚的心理 |
| 体态 | 低头、双手交握或抱头、身体前倾、肩背微缩 | 低头交握/抱头、身体前倾、肩背微缩——悔恨的姿态 |
| 皮肤 | 面部可能泛红（因情绪激动），眼周微红 | 面部微泛红、眼周微红——情绪激动的生理反应 |
| 服装 | 同日常状态，可能因长时间静坐导致衣服褶皱加深 | 同日常、褶皱因长时间保持一个姿势而加深 |

### 坚定/决心状态（报警/举报/指认时刻）

| 维度 | 描述 | 提示词 |
|---|---|---|
| 面容 | 面部线条清晰，下颌微抬或微收，表情克制有力 | 面部线条清晰、下颌微抬或微收——决心下定后的面容 |
| 眼神 | 目光笔直向前/看向执法者，不回避不躲闪，瞳孔稳定 | 目光笔直向前、稳定不躲闪——下定决心的注视 |
| 体态 | 坐姿/站姿端正——不再是含胸低头，而是肩背展开、重心稳定 | 坐姿端正、肩背展开——决心下定后的体态变化 |
| 皮肤 | 正常皮肤状态，但因情绪激动可能面部微红 | 面部微红——情绪激动但已下定决心的状态 |
| 服装 | 同日常状态——服装不变但体态变化带来不同的视觉效果 | 日常服装、但因体态变化呈现不同的视觉效果 |

## 三、服装变体

### 日常便服（基础变体）

素色基础款上衣+素色基础款下装，无品牌标识、无印花、无装饰。日常穿着状态，面料有自然微皱。

### 职业/工作服装（执法/公务人员变体）

执法人员：深蓝色/藏青色制服（警服），佩戴警徽/肩章/胸牌。整洁利落，面料挺括但有穿着后的自然褶皱。
其他职业：按角色职业设定——如保安制服、社区工作人员马甲、志愿者红马甲等。

### 特殊场合服装（针对不同叙事阶段）

- **受害前**：日常便服，状态整洁——正常生活的状态
- **受害中/被骗中**：同一套服装但随时间推移出现更多褶皱、可能被雨淋湿/汗浸——暗示时间的流逝和状态的恶化
- **受害后/被解救后**：可能换了一套干净的基础服装（家人或救助机构提供的），或者仍是原来的服装但已经清洗过——暗示救助的介入
- **法庭/执法场合**：整洁的基础服装，不过分正式也不随意——接受法律程序的状态

## 五、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有衍生变体必须保持同一人物的核心面容特征——颌骨轮廓、鼻梁转折、耳廓形状、发际线走向一致 |
| R2 | 情绪变化必须体现在面容、眼神、体态、皮肤状态四个维度的联动变化——不是换表情，是整个人的状态变化 |
| R3 | 服装变体必须保持"素色基础款"基调，无品牌标识、无印花、无装饰 |
| R4 | 必须指定「中灰无缝背景纸，无场景、无光斑、无渐变」 |
| R5 | 必须声明「真人实拍摄影 + 非3D非渲染非CG」核心锚定 |
| R6 | 必须声明「素颜无妆 + 真实皮肤质感 + 非磨皮」 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「不同变体的面容不一致——变成另一个人」 |
| X2 | 严禁「过度戏剧化的情绪表演——社会教育的情绪是真实微妙的，不是舞台剧」 |
| X3 | 严禁「3D渲染 / CG / 过度磨皮 / 硅胶感」等非真人质感 |
| X4 | 严禁「古风 / 非当代 / 赛博朋克 / 科幻」等非当代社会背景元素 |
| X5 | 严禁「通过服装变体美化/浪漫化受害过程或违法行为」 |
| X6 | 严禁「裸露 / 低俗 / 暴力血腥」等不当内容 |
