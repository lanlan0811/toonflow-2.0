# 真人实拍社会教育场景衍生 · 约束手册

---

## 一、场景衍生原则

> 社会教育场景的衍生不是"换一个背景"，而是"同一空间在不同社会情境下的状态变化"。核心原则：**空间不变，命运已变**。

1. **空间一致性优先** — 衍生场景必须是同一个空间的不同状态——同一间询问室在不同时间的状态、同一个家庭客厅在事件前后的对比
2. **光变驱动情变** — 场景衍生的核心驱动是"光的变化"——同一空间从暖色窗光到冷白顶光的切换，本身就是叙事
3. **状态变化有因果** — 场景的物理变化（杂物增多/减少、家具变动、陈设变化）必须有叙事逻辑支持
4. **社会教育叙事功能** — 每个衍生变体服务于"警示对比"——"之前"和"之后"的视觉对比是社会教育的核心叙事手段

## 三、天气变体

### 晴天

| 维度 | 描述 | 提示词 |
|---|---|---|
| 光源 | 日光直射或漫射，阴影清晰或柔和，色温正常 | 晴天自然光、阳光穿过窗户/树冠、清晰/柔和阴影 |
| 氛围 | 日常、正常——未被特殊天气打扰的普通一天 | 晴日日常、未被干扰的普通社会空间 |

### 阴天

| 维度 | 描述 | 提示词 |
|---|---|---|
| 光源 | 天光均匀漫射，无直射光，阴影极弱，色温偏冷（5500-6500K） | 阴天均匀漫射光、无直射阴影、冷调均匀照明 |
| 氛围 | 压抑/沉闷/停滞——适合低迷情绪、真相即将揭晓前的氛围 | 阴天的沉闷、暴风雨前的宁静、情绪的低谷 |

### 雨天

| 维度 | 描述 | 提示词 |
|---|---|---|
| 光源 | 天光被云层和雨幕散射，光线柔和偏冷，湿路面产生反射 | 雨天散射光、湿路面反射、窗玻璃雨痕、冷调柔和 |
| 氛围 | 悲伤/清洗/转折——雨水冲刷一切、适合情绪高潮和转折 | 雨中的反思、被雨水冲刷的城市、转折的时刻 |
| 核心元素 | 湿路面反射街灯/车灯光斑、窗玻璃上的雨痕、积水中的倒影 | — |

## 四、情绪变体

### 温馨日常

同一空间在温馨日常状态：窗光明亮柔和、室内暖光为主、陈设整洁但自然的日常状态、有生活温暖感

### 紧张/沉重

同一空间在紧张状态下：光源单一化（可能只留一盏灯）、阴影增多、陈设可能凌乱（翻过的抽屉/散落的文件）、空间显得比日常更空旷或更拥挤

### 希望/新生

同一空间在希望救赎状态下：逆光/窗光亮起、暖色调增强、空间中出现新的元素（一束花/一杯热水/一件外套——救助的痕迹）

## 六、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 所有衍生变体必须基于同一个核心空间——同一房间、同一街道、同一场所的不同状态 |
| R2 | 时段变化必须通过光源变化来驱动——色温、方向、光比的变化是最核心的衍生手段 |
| R3 | 天气变化必须通过光线质量和反射/散射的变化来呈现，而非后期特效 |
| R4 | 必须声明"真人实拍摄影 + 非3D非渲染非CG"核心锚定 |
| R5 | 严禁出现任何人物——"空的场景"本身就是叙事 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 严禁「不同变体使用了不同的空间——核心空间不一致」 |
| X2 | 严禁「3D渲染 / CG / 后期添加的天气特效 / 合成光效」 |
| X3 | 严禁「古风 / 非当代 / 赛博朋克 / 科幻」等非当代社会场景 |
| X4 | 严禁「出现人物 / 人影 / 人体轮廓」 |
| X5 | 严禁「样板间 / 全新无痕 / 无生活痕迹」的完美场景 |
