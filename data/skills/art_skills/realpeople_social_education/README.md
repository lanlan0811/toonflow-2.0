# 真人实拍社会教育短片 · 风格说明

本手册专为"真人实拍社会教育短片"题材视觉创作打造，所有美术提示词、视觉规范和生成内容严格限定于真人实拍级社会教育影像美学体系。

---

## 一、风格核心定义

**真人实拍社会教育短片**是以**真人影视摄影美学**为基准、以**社会教育与公益宣传**为核心的视觉风格体系。其核心特征为：

- **真人写实摄影质感**：以真人影视级摄影为锚点，画面呈现真实人物的皮肤纹理、毛发细节、微表情与肢体语言，杜绝3D建模感、CG渲染感、手绘感
- **社会教育叙事语境**：所有内容服务于社会教育目的——反邪教警示教育、反诈骗宣传、公共安全科普、公益道德传播、法制教育等，兼具警示性与人文关怀
- **纪实美学为主导**：以纪录片式的真实感为核心视觉追求，辅以电影级画面质感，强调"真实可信"高于"视觉炫目"
- **情感穿透力优先**：通过真实的面孔、真实的环境、真实的情感瞬间来传递教育信息，让观众在情感共鸣中接受教育内容
- **警示与温度并存**：在呈现社会问题的严肃性时不失人文温度，在警示教育中留有希望与正向引导

### 创作理念与核心价值

本手册的核心创作理念为 **"真实即力量，教育即关怀"**：

1. **真实感的警示教育价值**：社会教育短片的核心是"让人相信"。真人实拍的肌肤纹理、眼神温度、生活场景的真实感，是建立观众信任的基础。只有观众相信画面中的故事，教育信息才能被接收。

2. **社会议题的视觉化表达**：无论是反邪教、反诈骗还是公共安全教育，都需要将抽象的社会议题转化为具体可感的视觉叙事。真实的场景、真实的人物、真实的情感冲突，是连接抽象议题与观众情感的桥梁。

3. **警示而不恐吓的平衡**：社会教育短片需要在警示性与心理安全感之间取得平衡。画面足够真实让观众感受到问题的严重性，但不刻意制造恐怖或不适，避免产生抗拒心理。

4. **正向引导的叙事立场**：每一部社会教育短片都应有一个"出口"——在揭示问题之后给出希望、解决方案或行动指南。视觉上，这意味着从阴暗到光明、从混沌到清晰的视觉过渡。

**核心价值：**

- 为AI视觉创作者提供一套系统化、可复用的真人实拍社会教育短片提示词方法论
- 建立从教育概念到警示画面的标准化视觉创作管线
- 降低社会教育题材AI视觉创作的技术门槛与试错成本
- 填补当前AI视觉创作领域"真人实拍社会教育题材系统化方法论"的空白

## 三、视觉风格定位

### 3.1 整体风格定位

**"纪实美学为基底，电影级画面为辅助"**

以真人实拍的纪实感为核心视觉追求，电影级画面品质为辅助手段。整体风格在"警示教育"与"人文温度"之间寻找平衡——画面真实有力但不刻意煽情，警示性强但不制造恐慌，教育目的明确但不生硬说教。

### 3.2 影像质感层级

| 质感维度 | 定位标准 |
|---|---|
| **皮肤质感** | 真人摄影级，可见自然的皮肤纹理、毛孔、细纹、痣斑，杜绝过度磨皮与塑料感。社会教育题材尤其需要保留皮肤的"真实感"——这是可信度的基础 |
| **毛发质感** | 发丝自然蓬松或垂落，受光面与背光面层次清晰，杜绝僵硬发块。在紧张/憔悴场景中，发丝可呈现被汗浸湿或略微凌乱的状态 |
| **衣物质感** | 面料纹理可辨，褶皱符合身体动态与重力。社会教育题材的着装以日常便服为主，强调"大众化"而非"时尚感" |
| **环境质感** | 墙面有使用痕迹、地面有自然磨损、空间有生活气息。社会教育场景的"真实感"高于"美感"，避免过度美化的空间 |
| **光影质感** | 以自然光与实用光源为主，避免过度戏剧化的光影。纪实感的均匀照明优先于氛围感的强光比 |

## 四、关键视觉元素

### 4.1 人物视觉元素

社会教育短片的人物视觉以**"真实个体、社会群像"**为核心：

- **面容标准**：真人肌肤质感，五官自然立体。**素人感优先于明星感**——角色应该是"你每天在生活中会遇到的人"，而非"杂志封面上的人"。面容保留真实的不完美（痣、斑、不对称、岁月痕迹）
- **妆容标准**：极淡妆或素颜状态——无色唇膏、透明眉胶级别。女性角色不画眼线/不涂口红/不打粉底。在"受害后/困境中"场景中可呈现素颜带憔悴感
- **发型标准**：真实日常发型——自然黑发/深棕发，不染非自然色。发丝自然状态，可因场景需要呈现微乱/汗湿/被雨打湿等状态
- **服饰标准**：大众日常着装——T恤、衬衫、外套、卫衣、长裤等基础款。**无品牌标识、无大面积印花、无装饰性设计**。社会教育题材的着装强调"普适性"和"日常性"，不追求时尚感
- **表情与体态**：微表情真实细腻（眼神的躲闪/坚定/恐惧/悔恨），肢体语言自然不做作。警惕时的体态微缩、说谎时的眼神游移、受害后的低头含胸——这些细微体态是社会教育叙事的关键

### 4.2 场景视觉元素

- **家庭场景**：普通家庭的客厅/卧室/厨房——沙发茶几+电视、餐桌餐椅、窗帘窗台、生活杂物的自然散落。强调"普通家庭的日常状态"，不豪华不寒酸
- **社区/街道**：居民小区、社区公告栏、老城区巷弄、菜市场、社区活动中心——市井日常的烟火气，社会教育发生的最真实土壤
- **办公/学校**：普通写字楼办公室、学校教室/走廊/宿舍——揭示邪教/诈骗/不良信息可能渗透的日常场所
- **执法/救助机构**：派出所询问室（铁椅+日光灯+白墙+摄像头）、反诈中心（大屏幕+工位+实时数据）、社区调解室——正义与救助的视觉符号
- **警示空间**：拘留所/看守所内部（灰墙+铁门+日光灯冷白+狭小空间）、法庭（被告席/法官席+严肃氛围）——警示教育的终点站
- **城市公共空间**：地铁站/公交站/公园/天桥——"日常中的陷阱"叙事，展示邪教/诈骗可能发生的公共场所
- **特殊场所（反邪教专题）**：地下聚会点（昏暗+杂物+标语横幅）、洗脑讲座现场（投影+讲台+封闭空间）、脱离后的心理辅导室（温暖+私密+安全感）

### 4.3 道具与细节元素

- **警示道具**：反诈宣传单、报警回执、法院传票、拘留通知书、悔过书——法律与警示的视觉符号
- **通讯设备**：手机（诈骗短信/钓鱼链接界面、邪教群聊、反诈APP界面）、笔记本电脑（虚假投资平台、洗脑直播）、老年机（针对老年诈骗的典型道具）
- **生活道具**：钱包/银行卡（被盗刷/被骗后的残留）、身份证/户口本（身份盗用）、钥匙/门禁卡（家被侵入的暗示）
- **宗教/邪教道具**：邪教宣传册/传单、邪教标识饰品、洗脑教材/录音带、聚会用的蜡烛/横幅——需谨慎使用，服务于警示教育目的
- **情感符号道具**：家庭合影照片（幸福回忆vs破碎现在的对比）、孩子玩具（针对家庭受害者的情感冲击）、病历本/药瓶（健康诈骗/邪教拒医）
- **执法道具**：警察制服/警徽、手铐、执法记录仪、询问笔录——正义介入的视觉符号

### 4.4 镜头语言元素

| 景别 | 叙事功能 | 社会教育特征 |
|---|---|---|
| 大特写 | 情绪爆发点、关键细节揭示 | 眼神中的恐惧/悔恨/醒悟、手机诈骗界面的关键文字、证据的关键细节 |
| 近景 | 人物情感、对话反应 | 面部微表情、受害者的恐惧、执法者的严肃、家人的悲痛——社会教育的情感核心 |
| 中近景 | 人物互动、动作叙事 | 对话中的多人互动、询问/审讯过程、救助时刻 |
| 中景 | 人物关系、空间交代 | 人物在空间中的位置、家庭关系呈现、执法与被执法者的权力关系 |
| 全景 | 环境氛围、人物孤寂/困境 | 受害者的孤独处境、空荡的家、冷清的街道 |
| 空镜 | 情绪缓冲、隐喻留白 | 警示标语的空镜、夜晚空荡的街道、逐渐暗去的灯光——留给观众思考的空间 |

## 五、严禁内容（优先级最高）

本风格严格排除以下内容类型，确保视觉体系纯净统一：

1. **非真人质感**：3D渲染/CG建模/次世代游戏引擎/动画风格/手绘插画/水彩油画/二次元/Voxel/低多边形等一切非真人实拍影像质感
2. **非当代世界观**：古风/古装/仙侠/玄幻/武侠/民国/西方奇幻/中世纪/赛博朋克/科幻未来/星际太空等非当代社会教育背景
3. **非中国语境**：明显非中国社会特征的场景、建筑、标识、人物造型——社会教育内容必须扎根中国社会现实
4. **技术质量缺陷**：低分辨率/模糊/噪点/变形/畸形肢体/五官扭曲/比例失调/抠图感/塑料感/过度磨皮/假面感
5. **过度恐怖渲染**：过度血腥/暴力/恐怖画面——社会教育需要警示但不制造心理创伤。禁止刻意制造恐怖氛围、"鬼片"式光影
6. **不当内容**：暴露/低俗/暗示性/歧视性表达/政治敏感元素/违法内容
7. **过度美化犯罪/邪教/诈骗**：严禁以任何形式美化/浪漫化/神秘化邪教组织、诈骗行为、违法犯罪活动
8. **水印痕迹**：水印/文字/LOGO/签名/边框/AI生成工具痕迹

## 六、使用说明

### 提示词使用原则

1. **前缀先行**：所有生成内容必须首先加载"前缀提示词"中的全局风格锚定段，锁定真人实拍社会教育写实质感基线
2. **专项叠加**：在全局前缀基础上，按需叠加角色/场景/道具/分镜专项提示词
3. **负面必加**：每次生成必须附带"负面提示词"，规避伪影与风格偏差
4. **迭代优化**：建议首轮生成后根据效果对专项提示词做微调，而非修改全局前缀
5. **平台适配**：本手册提示词优先适配Seedance（视频生成）与Seedream（生图），部分提示词可能需根据平台特性微调
