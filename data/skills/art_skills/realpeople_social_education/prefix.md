## 二、摄影机语法（替代"渲染引擎"）

> 真人实拍社会教育不依赖渲染管线。画面品质取决于摄影机的选择、镜头的性格、光线的调度。以下为"摄影机语法"核心参数。

### 2.1 画幅与格式

| 参数 | 基准值 | 美学意图 |
|---|---|---|
| 画幅比 | 16:9（默认宽幅）或 9:16（手机短视频） | 警示教育适配主流传播平台；纪实横屏展现人与环境的关系；竖幅适合新媒体传播 |
| 影像格式 | 全画幅数字摄影 / 35mm胶片质感 | 真实细腻的影像质感，自然的暗角过渡，微妙的颗粒结构 |
| 色彩科学 | ARRI Alexa 类电影色彩科学 / 纪实自然色彩 | 肤色还原自然、高光滚降柔和、暗部不压死且有层次；纪实场景可选去饱和自然色调 |

### 2.2 镜头性格

| 焦段 | 叙事性格 | 典型应用 |
|---|---|---|
| 24-28mm（广角） | 环境包围人物、纪实感、揭露全貌 | 审讯室空镜、家庭全景、社区环境、警示空间 |
| 35mm（小广角） | 主观视点、身临其境、最接近人眼的纪实感 | 纪实跟随、家庭对话、街头走访、询问过程 |
| 50mm（标准） | 人眼等价、客观注视、不美化不扭曲 | 人物肖像、受害者/违法者表情记录、对话中的情感凝视 |
| 85mm（中长焦） | 压缩空间、拉近距离、捕捉不易察觉的微表情 | 特写捕捉、远距离观察、情绪爆发的瞬间 |
| 135mm+（长焦） | 极端压缩、窥视感、将人物从环境中剥离 | 暗中观察/跟踪视角、城市中孤立个体的远眺、监视感 |

### 2.3 机位与运动

| 手法 | 视觉心理 | 使用场景 |
|---|---|---|
| 固定机位 | 冷静、客观、让事件自己说话 | 对话、询问、审讯、空镜留白——社会教育的基础镜头语言 |
| 手持微晃 | 在场感、呼吸感、纪实介入 | 情绪波动场景、跟踪/慌乱/冲突时刻、街头纪实跟随 |
| 斯坦尼康/稳定器 | 平滑流动、揭示全貌、逐步深入 | 空间展示、人物入场、从外部进入内部（如从街道进入派出所） |
| 俯拍 | 命运感、压迫、审视 | 审讯俯视、受害者蹲坐、桌面证据的俯视记录 |
| 仰拍 | 压迫感或崇高感 | 执法人员的权威感、建筑/警示牌的崇高感 |
| 焦点偏移（Rack Focus） | 注意力的转移、真相的揭示 | 从人物面部移至背后的证据/线索、从模糊到清晰的真相浮现 |

## 三、社会教育色谱——光与材料的真实色彩

> 真人实拍社会教育的色彩来自中国真实社会空间的光与材料，不是引擎色板。以下色盘提取自：派出所日光灯的白、家庭客厅窗光的暖、审讯室冷白墙壁的肃穆、社区街道夕阳的金、深夜街灯的黄。

### 3.1 社会教育光源色谱

| 光源 | 色温（K） | 光色描述 | 社会教育场域 |
|---|---|---|---|

| 上午天光 | 5500-6000 | 冷白清澈，阴影锐利，建筑表面材质感最强 | 执法机构室内、社区走访、家庭日间日常 |
| 正午顶光 | 6000-6500 | 冷硬强光，阴影短而浓重 | 街头/公园中的相遇——邪教/诈骗发生的公共场合 |

## 四、全局约束规则

### 4.1 影像真实性约束（必守）

| 编号 | 规则 |
|---|---|
| P1 | 必须以「真人摄影 / 真人实拍 / 真人纪实影像」为生成锚——画面必须读作"摄影机拍下的"，而非"计算机生成的" |
| P2 | 必须指定一个具体的光源逻辑：这场戏的光从哪里来？（窗/灯/天空/屏幕/荧光灯管）光源方向、色温、软硬须自洽 |
| P3 | 必须保留人物皮肤的真实质感——毛孔、细纹、痣、自然肤色不均、眼周微暗、T区自然油光——杜绝塑料磨皮与AI感假面 |
| P4 | 发丝必须呈现自然状态——日常状态、被风吹乱、被雨打湿、因紧张出汗而微粘额头——杜绝建模感整齐发块 |
| P5 | 服饰必须体现"穿过"的痕迹——面料褶皱、领口自然变形、日常磨损——杜绝全新样板衣感。社会教育角色的服装应具有"大众日常感" |
| P6 | 场景必须有"被使用过"的证据——桌面物品自然散落、墙面有生活痕迹、地面不完美——社会教育场景的真实感是可信度的基础 |
| P7 | 人物必须处于自然体态——真实的站姿（重心偏一侧）、不刻意的表情、日常的手部动作。在社会教育叙事中，体态的"真实"比"好看"重要 |
| P8 | 必须指定一个具体的中国社会空间——拒绝"generic city"或任何无国别标识的架空场景。社会教育必须扎根中国社会现实 |

### 4.2 摄影品质约束（必守）

| 编号 | 规则 |
|---|---|
| Q1 | 必须声明一种摄影机的"在场方式"：固定注视 / 手持纪实 / 稳定器流动 / 长焦远观 |
| Q2 | 景深必须具有叙事意图——浅景深突出人物情绪、深景深强调环境全貌、焦点偏移引导注意力到关键信息 |
| Q3 | 高光不得过曝死白，暗部不得死黑无层次——保留高光滚落和阴影细节是纪实电影感的基础 |
| Q4 | 必须声明画面的颗粒质感偏好：数字摄影干净锐利（默认）/ 纪实风格微颗粒（可选） |
| Q5 | 色彩调性必须与教育叙事情绪匹配——警示不溢色、温情不甜腻、纪实不灰败 |

### 4.4 风格越界判定

> 当你生成的画面出现以下任一信号，说明已经偏离真人实拍社会教育风格：

| 越界信号 | 说明 |
|---|---|
| 人物皮肤像硅胶 | 磨皮过度，失去真人质感——立即停用"光滑""完美肌肤"类词汇 |
| 光源来源不明 | 画面亮但说不清光从哪来——补全光源逻辑，拒绝"全局均匀亮" |
| 场景没有中国特征 | 空间失去中国社会辨识度——加入中国家庭/社区/执法机构的典型元素 |
| 画面过于干净 | 场景像刚装修完的样板间——加入生活痕迹、使用痕迹、岁月痕迹 |
| 人物站姿像模特 | 体态僵硬、对称、摆拍感——给一个自然的身体重心偏移 |
| 色彩像滤镜APP | 色调统一但廉价——回归光源色温驱动，不要"滤镜套用" |
| 警示变恐怖 | 对违法犯罪/邪教的展示变成恐怖片渲染——记住这是社会教育，不是恐怖片 |
