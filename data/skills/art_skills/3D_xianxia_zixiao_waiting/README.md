# 3D仙侠短剧《紫霄浮岛·七日等候》视觉手册

本手册服务于 `data/shot-list/01_storyboard.md` 的 82.8 秒短剧片段。故事发生在紫霄剑宗浮岛：墨秋霜、夏浅浅、南宫离与红衣三师妹到山门等候虾仁，确认岛上无人后分头寻找，最终留下三人原地等待七天。本包是本片段的唯一视觉基线，优先保证角色、浮岛空间和“空无一人”的悬念连续。

## 1. 项目视觉定位

- **类型**：东方仙侠、宗门群像、轻冲突、悬念式结尾。
- **媒介质感**：高精度国风 3D 渲染，电影级构图与体积光；不是照片、真人摄影或 Q 版动画。
- **人物观感**：成年亚洲（东亚）面部特征，正常肩宽和手部比例，七头身左右；表演依靠眼神、呼吸、衣摆和微小重心变化。
- **画幅与镜头**：16:9；默认 24fps；镜头运动平稳、克制，允许缓推、缓拉、横移、轻环绕和航拍，不使用炫技穿越。
- **色彩基线**：月白、青灰、青绿为大面积基底，冰蓝/淡紫/白绿/赤红用于角色识别，金色仅作发饰与法器边缘点缀。
- **光线基线**：清晨柔光贯穿主体段落，薄雾与云海形成空气透视；第 17—18 镜用晨昏变化叠化表达七日，不改变建筑方位。

## 2. 必须保持的叙事事实

1. 虾仁在本片段中始终不出镜，不生成他的正面、背影、剪影或替身。
2. 浮岛应“整洁但无人活动”，不能变成废墟，也不能补入杂役、围观者或额外弟子。
3. 四名女子的横排站位、服装色块和身高层级跨镜头稳定；南宫离在第 14 镜离场后不再出现在画面中。
4. 建筑、山门、石阶、候客平台、主殿和瀑布云雾的相对方位固定，只改变机位和景别。
5. “两个月旧怨”通过红衣三师妹的神情和对白体现；“七天等待”通过时间流逝和姿态坚持体现，不用字幕解释。

## 4. 角色设定图：四视图硬规范

每个角色必须输出同一画面、同一比例的四视图，不得把四张图拼成不同角色或不同服装：

1. **正面 0°**：全身，头顶至脚底完整入画，双臂自然下垂。
2. **三分之二视图 45°**：全身，展示鼻梁、肩线、袖型和披帛层次。
3. **侧面 90°**：全身，明确发冠/发尾、鼻唇轮廓、衣摆厚度。
4. **背面 180°**：全身，明确后发、背部纹样、披风和裙摆结构。

四视图统一规则：亚洲（东亚）面部特征、成年比例、七头身、墨黑发色、同一套服装与配饰；中性微表情；素灰背景；均匀柔光；无文字、无水印、无额外人物。人物不可幼态化、不可 Q 版化、不可出现现代妆容或现代服饰。

## 5. 道具设定图：五视图硬规范

片中道具以克制、可叙事为原则，核心为南宫商会传讯玉牌；石台、山门匾额和护岛结界属于场景组成，不另造与剧情无关的奇观道具。每件道具必须同画面输出五视图：

1. 正面 0°；2. 三分之二视图 45°；3. 侧面 90°；4. 背面 180°；5. 顶视/底部结构视图（根据道具形态选择其一）。

五视图共用同一模型、材质、尺寸和磨损状态；素灰背景；均匀柔光；道具独立陈列，不能被手持、佩戴或使用；画面不能出现人物、手部、人体影子或文字。玉牌使用温润青白玉、细金线刻南宫商会纹章、边缘有可控的灵力微光；激活态只增加一圈低强度青绿色符纹，不改模型结构。

## 6. 场景设定图：五视图硬规范

每个场景资产必须输出同一空间的五视图，五视图不是拼贴风景，而是用于锁定空间连续性的环境设计表：

1. **英雄主视图**：人眼平视，候客平台与山门长阶形成主轴。
2. **左侧视图**：展示平台边缘、云海落差、廊道入口。
3. **右侧视图**：展示主殿飞檐、护岛结界和瀑布云雾。
4. **背向视图**：从山门方向回看四人将站立的候客平台及远山。
5. **俯视平面视图**：标明平台、石阶、正殿、廊道、结界边界和云海落差的相对位置。

场景图禁止出现人物、人影、人体轮廓、现代建筑和网格/分屏；每格都要能辨认同一套建筑结构。场景必须有前景（石栏/落叶/衣摆可不出现人物）、中景（平台/石阶/殿门）、后景（浮岛、远山、云海），并保留整洁、无人、轻微使用痕迹的质感。

## 7. 浮岛空间与材质

- **候客平台**：浅灰青石，边缘有细微风化和薄青苔；四人横排时中央留出墨秋霜的主轴。
- **山门长阶**：连续长石阶通向关闭的正殿，阶面干净但没有新近清扫痕迹；用于第 02、07、08、16 镜。
- **主殿**：青灰瓦、月白石基、青绿色铜饰，门窗紧闭，不放人物或灯火。
- **护岛结界**：极薄、半透明、青白色的环形气层，只在大远景中可读；不得做成霓虹或科幻力场。
- **云海与远山**：月白云层、青灰远山、低对比空气透视；云雾有缓慢流动，避免纯静态背景。
- **材质总则**：PBR 石材、木材、丝绸、玉石和金属；自然磨损可见但不脏乱，不使用塑料感、贴图模糊或过度锐利的 CG 外观。

## 8. 色彩、光影与特效

| 段落 | 主色 | 光影/特效 | 情绪用途 |
|---|---|---|---|
| 01—04 开场与空岛 | 月白、青绿、青灰 | 清晨柔光、薄雾、低强度体积光 | 交代地点并制造“无人回应” |
| 05—10 旧怨与判断 | 月白、淡紫、赤红点缀 | 阴影略收、回声渐远、空气涟漪极轻 | 红衣急躁与夏浅浅理性形成对比 |
| 11—16 决策与分头 | 冰蓝、白绿、赤红 | 稳定侧光、衣摆和披帛有风动 | 墨秋霜定调，南宫离获得动作出口 |
| 17—18 七日等待 | 月白→淡金→靛蓝→月白 | 云层、日影、星色叠化；不改变空间模型 | 以时间流逝和静止姿态收束悬念 |

灵力表现仅用于墨秋霜传音、南宫离传讯和护岛结界：采用空气轻微波纹、玉牌低亮符纹、薄雾折射；禁用爆炸、闪电、霓虹线条、大面积粒子和西式魔法符文。

## 9. 全局禁止项

- 禁止写实摄影、照片感、真人影视皮肤；本包是国风 3D 角色与场景渲染。
- 禁止西方奇幻、赛博朋克、科幻、现代建筑、现代服饰、现代道具。
- 禁止 Q 版、卡通、动漫、二次元、夸张表情和夸张肢体。
- 禁止加入虾仁、杂役弟子、围观群众或临时背景人物。
- 禁止改变四人服装主色、发型、发冠和相互站位。
- 禁止把五视图生成成五个不同资产；禁止透视、比例、光向和建筑轴线跳变。
- 禁止字幕、水印、标题叠字、UI 文字；山门匾额若出现只保留不可读的古朴纹样。
