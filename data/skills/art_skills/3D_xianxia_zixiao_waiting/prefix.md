# 全局视觉锚定 · 3D仙侠短剧《紫霄浮岛·七日等候》

本前缀会与角色、道具、场景和分镜提示词一起使用。它优先锁定短剧的空间连续性、亚洲人物特征和视图数量。

## 必须包含的风格锚定

3D仙侠渲染，高精度建模，PBR材质，国风3D，东方美学，电影级光影，体积光，环境光遮蔽，景深层次，细腻布料与发丝，月白—青绿—青灰色彩基线，清晨薄雾云海，16:9 电影构图（设定图按对应视图比例）。

含人物时必须补充：成年亚洲/东亚面部特征，正常人体比例，七头身左右，面容与服饰跨视图一致，古风仪态，中性微表情。

含道具时必须补充：纯道具独立陈列、五视图、同一模型同一材质、无人物和手部。

含场景时必须补充：同一空间五视图、前景/中景/后景、空气透视、无人环境、空间轴线固定。

## 本片专属连续性词

紫霄剑宗浮岛，候客平台，山门长阶，关闭的正殿，青灰瓦与月白石基，薄青色护岛结界，层叠远山，月白云海，清晨柔光，风吹衣摆，空旷无人，回声消散，七日时间流逝。

## 角色锚定词

墨秋霜：冰蓝白衣、黑金高发冠、白色披肩、冷静公允、居中主位。

夏浅浅：淡紫长裙、银紫披帛、温柔机敏、观察空廊、耐心等待。

南宫离：白绿金服、英气干练、南宫商会腰牌/传讯玉牌、动作果断、快速离场。

红衣三师妹：赤红长衣、红金发簪、抱臂皱眉、语速偏快、旧怨不耐。

## 生成硬规则

1. 角色设定图只生成四视图：正面 0°、三分之二 45°、侧面 90°、背面 180°，全身头顶至脚底完整入画。
2. 道具设定图只生成五视图：正面、三分之二、侧面、背面、顶视或底部结构；同一画面同一道具，不出现手持/佩戴。
3. 场景设定图只生成五视图：英雄主视图、左侧、右侧、背向、俯视平面；同一建筑结构，不出现人物。
4. 设定图背景为素灰或中性浅灰；镜头分镜背景必须来自紫霄浮岛，不得改成其他仙山。
5. 画面无字幕、无水印、无标题叠字、无 UI 文字；山门匾额文字不要求可读。
6. 维持 16:9 叙事画幅；不使用 Q 版头身比、卡通线稿、照片感或现代元素。

## 禁止词与负向约束

避免：照片级真实、写实摄影、卡通、动漫、二次元、Q版、西方奇幻、赛博朋克、科幻、现代建筑、现代服饰、荧光霓虹色、塑料质感、低精度建模、粗糙贴图、额外人物、虾仁出镜、字幕、水印、标题叠字。

## 推荐提示词骨架

`风格锚定 + 资产名/角色身份 + 视图数量与角度 + 形体/服饰/材质 + 场景与空间关系 + 情绪/动作 + 光影与色彩 + 一致性与禁止项`

输出提示词正文时，不要在画面中添加解释性文字，不要把视图数量改成模型自行猜测的拼图数量。
