# 分镜表约束 · 3D仙侠短剧《紫霄浮岛·七日等候》

## 1. 表格基础

- 画幅 16:9，单镜时长沿用 `01_storyboard.md` 的 1.0—7.8 秒范围，总时长约 82.8 秒；实际生成可前后微调 0.1—0.3 秒。
- 每行必须同时写清：镜号、时长、景别、机位/运镜、人物在画面中的位置、动作/微表情、场景空间、台词、声音、连续性备注。
- 画面描述优先于风格词，必须完整写出人物、道具和空间关系；不能只写“古风仙侠氛围”。

## 2. 景别与运镜策略

| 叙事任务 | 优先景别 | 允许运镜 |
|---|---|---|
| 建立地点和群像 | 全景、中远景、大远景 | 缓慢横移、缓推、航拍推进 |
| 读表情与立场 | 中景、中近景、近景 | 固定、极轻推、轻横移 |
| 无人证据 | 双人中近景带空廊、空阶 | 轻微横移展示空景 |
| 时间流逝与收尾 | 全景 | 缓慢后拉、极慢拉远、叠化 |

禁止甩镜、急推、手持晃动、旋转转场和穿越建筑。只有第 14 镜允许一次短促跟摇跟随南宫离离场。

## 3. 环境动态与画面呼吸

每 3—4 镜至少安排一项低强度环境动态：云海翻涌、薄雾散开、衣摆/披帛飘动、风铃轻响、落叶滚动、护岛结界气层微动。动态不能引入新角色或新的剧情信息。

## 4. 连贯性校验

- 01—04：四人都在，墨秋霜居中，镜头从群像逐步转向空岛。
- 05—13：四人仍在候客平台；夏浅浅和红衣三师妹的反应镜不能让墨秋霜消失。
- 14：南宫离从画面中离开，需留下空位/衣摆余风/玉牌声中的至少一项痕迹。
- 15—16：只有墨秋霜、夏浅浅、红衣三师妹，三人站在山门石阶前。
- 17—18：三人盘膝等待，光线与云层变化，虾仁仍不现身。

## 5. 台词、声音和视觉的配合

- 传音台词必须配远去回声，但回声消失后不能追加脚步或人声。
- 红衣台词的急躁通过语速、抱臂、甩袖和轻拨弦体现，不用夸张音效。
- 南宫离离场使用玉牌轻响、衣摆破风声和短促配乐落点；离场后恢复古琴/风声。
- “七天”只由旁白和时间叠化共同表达，画面不得出现现代日历、时钟、字幕或时间码。
